Class FitEllipsoid

  • java.lang.Object
    • org.bonej.geometry.FitEllipsoid

  • public final class FitEllipsoid
extends Object
    Ellipsoid fitting methods. Both rely on eigenvalue decomposition, which fails if the input matrix is singular. It is worth enclosing calls to yuryPetrov and liGriffiths in a try{FitEllipsoid.yuryPetrov} catch(RuntimeException re){} to gracefully handle cases where the methods cannot find a best-fit ellipsoid.
    Author:
    Michael Doube

      • yuryPetrov

        public static Object[] yuryPetrov(double[][] points)
        Ellipsoid fitting method by Yury Petrov.

        Fits an ellipsoid in the form Ax2 + By 2 + Cz2 + 2Dxy + 2Exz + 2 Fyz + 2Gx + 2Hy + 2Iz = 1 To an n * 3 array of coordinates.

        See MATLAB script

        Parameters:
        points - array[n][3] where n > 8
        Returns:
        Object[] array containing the centre, radii, eigenvectors of the axes, the 9 variables of the ellipsoid equation and the EVD
        Throws:
        IllegalArgumentException - if number of coordinates is less than 9
