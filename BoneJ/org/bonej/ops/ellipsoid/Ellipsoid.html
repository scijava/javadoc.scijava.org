org.bonej.ops.ellipsoid

Class Ellipsoid

  • java.lang.Object
    • org.bonej.ops.ellipsoid.Ellipsoid

  • public class Ellipsoid
extends Object
    A class that stores the properties of an ellipsoid.

    The class doesn't allow degenerate cases like a disc where one or more of the radii is zero.

    Author:
    Richard Domander, Alessandro Felder

        public Matrix4d getOrientation()
        Gets a copy of the orientation unit vectors of the ellipsoid.

        The orientations are the column vectors of the matrix.

        NB the vectors may form a left-handed basis! This may cause exceptions if the matrix is used with other linear algebra libraries.

        Returns:
        orientations of the semi-axes in homogeneous coordinates.

      • setSemiAxes

        public void setSemiAxes(Vector3d u,
                        Vector3d v,
                        Vector3d w)
                 throws IllegalArgumentException,
                        NullPointerException
        Sets the semi-axes of the ellipsoid.

        Semi-axes will be sorted by length in ascending order so that the shortest vector becomes the 1st and the longest the 3rd column vector in the orientation matrix. Thus radius a will be the length of the 1st column vector etc.

        Parameters:
        u - a semi-axis of the ellipsoid.
        v - a semi-axis of the ellipsoid.
        w - a semi-axis of the ellipsoid.
        Throws:
        IllegalArgumentException - if the vectors are not orthogonal.
        NullPointerException - if any of the vectors is null.
