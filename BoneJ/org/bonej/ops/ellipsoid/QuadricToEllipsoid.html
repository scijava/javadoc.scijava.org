    public class QuadricToEllipsoid
extends AbstractUnaryFunctionOp<Matrix4d,Ellipsoid>
implements Contingent
    Tries to create an Ellipsoid from a general equation of a quadratic surface i.e. quadric.

    The input equation must be in a matrix. If the quadric's polynomial in homogeneous coordinates (w = 1) is Ax2 + By2 + Cz2 + 2Dxy + 2Exz + 2Fyz + 2Gx + 2Hy + 2Iz, then the matrix must be:

     [a, d, e, g]
 [d, b, f, h]
 [e, f, c, i]
 [g, h, i, -1]
