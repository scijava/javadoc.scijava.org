    Calculate degree of anisotropy using the mean intercept length method. Samples vectors in a sphere; the location of the sphere is random within the stack. New sampling spheres are generated until the coefficient of variation of the anisotropy result reduces to a user-set threshold or a maximum number of spheres has been sampled.

    Author:
    Michael Doube
    See Also:

    Harrigan TP, Mann RW (1984) Characterization of microstructural anisotropy in orthotropic materials using a second rank tensor. J Mater Sci 19: 761-767. doi:10.1007/BF00540446 .

        Calculate degree of anisotropy for a binary stack, running until a stable result is achieved, or the maximum number of iterations occurs.
        Parameters:
        imp - ImagePlus input. A binary stack is required.
        minSpheres - minimum number of iterations
        maxSpheres - maximum number of iterations
        nVectors - number of vectors in the sampling sphere
        radius - radius of the sampling sphere
        vectorSampling - distance between samples along the sampling vector. Set to 2.3 * maximum voxel dimension for a safe and efficient sampling increment
        tolerance - coefficient of variation of results at which we accept result is stable
        doPlot - set to true if you want to see a plot of anisotropy versus number of repeats, updated in real time
        Returns:
        Object array containing degree of anisotropy, coordinates of rose plot and Eigenvalue decomposition (fabric tensor)
