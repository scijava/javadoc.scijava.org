        Returns the common unit string, e.g. "mm3" that describes the elements in the space.

        The common unit is the unit of the first spatial axis if it can be converted to the units of the other axes.

        Type Parameters:

        unitService - an UnitService to convert axis calibrations.
        exponent - an exponent to be added to the unit, e.g. '³'.
        Returns:
        the unit string with the exponent.
