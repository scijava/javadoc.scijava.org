Class Align_Image

  • java.lang.Object
    • Align_Image
  • All Implemented Interfaces:
    PlugIn

    public class Align_Image
extends Object
implements PlugIn
    Select two images with a Line ROI in each, and rotate/translate/scale one to the other.

    Stacks are not explicitly supported, but a macro can easily use this plugin for the purpose by iterating over all slices.

    Author:
    Johannes Schindelin, Michel Teussink
