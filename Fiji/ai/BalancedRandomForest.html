  • All Implemented Interfaces:
    Serializable, Cloneable, weka.classifiers.Classifier, weka.core.BatchPredictor, weka.core.CapabilitiesHandler, weka.core.CapabilitiesIgnorer, weka.core.CommandlineRunnable, weka.core.OptionHandler, weka.core.Randomizable, weka.core.RevisionHandler

    public class BalancedRandomForest
extends weka.classifiers.AbstractClassifier
implements weka.core.Randomizable
    This class implements a Balanced Random Forest classifier, it is an ensemble classifier of random trees where all classes have the same representation in the training process. Class for constructing a balanced forest of random trees.

    For more information see:

    Leo Breiman (2001). Random Forests. Machine Learning. 45(1):5-32.
    BibTeX: 
     @article{Breiman2001,
    author = {Leo Breiman},
    journal = {Machine Learning},
    number = {1},
    pages = {5-32},
    title = {Random Forests},
    volume = {45},
    year = {2001}
 }
    *

    Valid options are: 

     -I <number of trees>
  Number of trees to build.
     -K <number of features>
  Number of features to consider (<1=int(logM+1)).
     -S
  Seed for random number generator.
  (default 1)
     -D
  If set, classifier is run in debug mode and
  may output additional info to the console
