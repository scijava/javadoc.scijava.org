        Write an image to a chunked mipmap representation.
        Type Parameters:
        T - Pixel type
        D - Dataset handle
        Parameters:
        img - the image to be written.
        type - instance of the pixel type of the image.
        mipmapInfo - contains for each mipmap level of the setup, the subsampling factors and block sizes.
        io - writer for image blocks.
        executorService - ExecutorService where block-creator tasks are submitted.
        numThreads - How many block-creator tasks to run in parallel. (This many tasks are submitted to the @code
        loopbackHeuristic - heuristic to decide whether to create each resolution level by reading pixels from the original image or by reading back a finer resolution level already written to the hdf5. may be null (in this case always use the original image).
        afterEachPlane - this is called after each "plane of blocks" is written, giving the opportunity to clear caches, etc. may be null.
        progressWriter - completion ratio and status output will be directed here. may be null.
        Throws:
