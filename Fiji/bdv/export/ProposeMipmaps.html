Class ProposeMipmaps

  • java.lang.Object
    • bdv.export.ProposeMipmaps

  • public class ProposeMipmaps
extends Object
    Propose number of mipmap levels, as well subsampling factors and chunk size for each level.

    Choice of proposed chunksize is not based on any hard benchmark data currently. Chunksize is set as either 16x16x16 or 32x32x4 depending on which one is closer to isotropic. It is very likely that more efficient choices can be found by manual tuning, depending on hardware and use case.

    Tobias Pietzsch <tobias.pietzsch@gmail.com>
