bdv.img.hdf5

Class Partition

  • java.lang.Object
    • bdv.img.hdf5.Partition

  • public class Partition
extends Object
    Partition describes one partition file of a dataset that is split across multiple hdf5 files. An aggregating hdf5 linking the partition files can be created using the Partition information (without looking at the constituent files).
    Author:
    Tobias Pietzsch <tobias.pietzsch@gmail.com>

      • split

        public static ArrayList<Partition> split(List<TimePoint> timepoints,
                                         List<? extends BasicViewSetup> setups,
                                         int timepointsPerPartition,
                                         int setupsPerPartition,
                                         String basename)
        Split a sequence into partitions, each containing a specified number of timepoints and setups.
        Parameters:
        timepoints - list of all TimePoints.
        setups - list of all BasicViewSetups.
        timepointsPerPartition - how many timepoints should each partition contain (if this is ≤0, put do not split timepoints across partitions).
        setupsPerPartition - how many setups should each partition contain (if this is ≤0, put do not split setups across partitions).
        basename - This is used to generate paths for the partitions. Partitions are named "basename-TT-SS.h5" where TT and SS are the index of the timepoint and setup batch, respectively.
        Returns:
        list of partitions.
