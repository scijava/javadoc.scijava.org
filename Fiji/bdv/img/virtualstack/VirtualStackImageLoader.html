implements ViewerImgLoader, TypedBasicImgLoader<T>
    ImageLoader backed by a ImagePlus. The ImagePlus may be virtual and in contrast to the imglib2 wrappers, we do not try to load all slices into memory. Instead slices are stored in VolatileGlobalCellCache. Use createFloatInstance(ImagePlus), createUnsignedByteInstance(ImagePlus) or createUnsignedShortInstance(ImagePlus) depending on the ImagePlus pixel type. When loading images (getSetupImgLoader(int), BasicSetupImgLoader.getImage(int, ImgLoaderHint...)) the provided setup id is used as the channel index of the ImagePlus, the provided timepoint id is used as the frame index of the ImagePlus.
