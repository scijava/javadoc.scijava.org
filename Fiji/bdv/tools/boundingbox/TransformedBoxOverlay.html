Class TransformedBoxOverlay

  • java.lang.Object
    • bdv.tools.boundingbox.TransformedBoxOverlay
  • All Implemented Interfaces:
    OverlayRenderer, TransformListener<AffineTransform3D>

    public class TransformedBoxOverlay
extends Object
implements OverlayRenderer, TransformListener<AffineTransform3D>
    A BDV OverlayRenderer showing a TransformedBox. Displays a wireframe box and/or intersection with current viewer slice, with various adjustable parameters.
