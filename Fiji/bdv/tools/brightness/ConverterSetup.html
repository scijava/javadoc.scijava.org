bdv.tools.brightness

Interface ConverterSetup

  • All Known Implementing Classes:
    BigWarpConverterSetupWrapper, PlaceHolderConverterSetup, RealARGBColorConverterSetup

    public interface ConverterSetup
    Modify the range and color of the converter for a source. Because each source can have its own converter, this is used to adjust brightness, contrast, and color of individual sources.
    Author:
    Tobias Pietzsch <tobias.pietzsch@gmail.com>
