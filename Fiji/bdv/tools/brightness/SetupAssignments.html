Class SetupAssignments

  • java.lang.Object
    • bdv.tools.brightness.SetupAssignments

  • public class SetupAssignments
extends Object
    Manage a (fixed) set of ConverterSetups and (changing) set of MinMaxGroups, such that the following is always true:
    1. Every setup is assigned to exactly one group.
    2. No group is empty.
    Author:
    Tobias Pietzsch <tobias.pietzsch@gmail.com>

        Add the specified setup to the specified group. The setup is removed from its previous group. If this previous group is made empty by this, it is removed from the list of groups.

      • removeSetupFromGroup

        public boolean removeSetupFromGroup(ConverterSetup setup,
                                    MinMaxGroup group)
        Remove the specified setup from the specified group. If this group would be made empty by this, it is not removed from the group. Otherwise, after being removed, a new group is created containing only the specified setup, and this new group is added to the list of group. The settings of the new group are initialized with the settings of the old group.
        Returns:
        Whether or not removal was successful (so the corresponding checkbox can be re-checked)
