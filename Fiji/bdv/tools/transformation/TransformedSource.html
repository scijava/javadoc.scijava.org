bdv.tools.transformation

Class TransformedSource<T>

  • java.lang.Object
    • bdv.tools.transformation.TransformedSource<T>
  • Type Parameters:
    T - the type of the original source.
    All Implemented Interfaces:
    MipmapOrdering, Source<T>

    public class TransformedSource<T>
extends Object
implements Source<T>, MipmapOrdering
    A Source that wraps another Source and allows to decorate it with an extra AffineTransform3D.

    This extra transformation is made to capture manual editing of the actual transform in the SpimViewer.

    Author:
    Jean-Yves Tinevez - Sept 2013
