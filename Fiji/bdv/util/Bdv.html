bdv.util

Interface Bdv

  • All Known Implementing Classes:
    BdvHandle, BdvHandleFrame, BdvHandlePanel, BdvOverlaySource, BdvPointsSource, BdvSource, BdvStackSource, BdvVirtualChannelSource

    public interface Bdv
    Something that has a BdvHandle. This includes BdvSources, as well as BdvHandles (which return themselves with getBdvHandle()).

    Having a Bdv is useful for adding more stuff to a BigDataViewer window or panel. This is done using BdvFunctions.show(..., Bdv.options().addTo(myBdv)).
