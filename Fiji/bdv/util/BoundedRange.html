Class BoundedRange

  • java.lang.Object
    • bdv.util.BoundedRange

  • public final class BoundedRange
extends Object
    A bounded range with minBound <= min <= max <= maxBound.

    BoundedRang is immutable.

    withMin(double), withMinBound(double) etc derive a new BoundedRange with the given min, minBound etc, while maintaining the minBound <= min <= max <= maxBound property. For example, this.withMin(x), will also update max, if x > this.getMax().

    join(other) will derive a new BoundedRange, with min the minimum of this.getMin() and other.getMin() and so on.

    Author:
    Tobias Pietzsch
