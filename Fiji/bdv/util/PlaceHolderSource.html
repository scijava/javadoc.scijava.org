Class PlaceHolderSource

  • java.lang.Object
    • bdv.util.PlaceHolderSource
  • All Implemented Interfaces:
    Source<Void>

    public final class PlaceHolderSource
extends Object
implements Source<Void>
    A dummy Source that represents a BdvOverlay.

    When a BdvOverlay is shown (with BdvFunctions.showOverlay(BdvOverlay, String, BdvOptions)), a dummy Source and ConverterSetup are added to the BigDataViewer such that the visibility, display range, and color for the overlay can be adjusted by the user, like for a normal Source.

    PlaceHolderSource is not present at any time point, so it is never actually visible.

    Tobias Pietzsch
