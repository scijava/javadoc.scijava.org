bdv.viewer

Class BasicViewerState

  • java.lang.Object
    • bdv.viewer.BasicViewerState
  • All Implemented Interfaces:
    ViewerState

    public class BasicViewerState
extends Object
implements ViewerState
    Maintains the BigDataViewer state and implements ViewerState to expose query and modification methods. ViewerStateChangeListeners can be registered and will be notified about various state changes.

    This class is not thread-safe.

    Author:
    Tobias Pietzsch
