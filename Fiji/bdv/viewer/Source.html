Interface Source<T>

  • All Known Implementing Classes:
    AbstractSource, AbstractSpimSource, GridSource, JacobianDeterminantSource, PlaceHolderSource, RandomAccessibleIntervalMipmapSource, RandomAccessibleIntervalSource, RandomAccessibleIntervalSource4D, RandomAccessibleSource, RandomAccessibleSource4D, RealRandomAccessibleIntervalSource, RealRandomAccessibleSource, RenamableSource, SpimSource, TransformedSource, VolatileRandomAccessibleIntervalMipmapSource, VolatileSpimSource, WarpedSource, WarpMagnitudeSource

    public interface Source<T>
    Provides image data for all timepoints of one view setup. Note that (partly for convenience and partly for historical reasons) the time-point index is used here instead of the timepoint id. This timepoint index is an index into the ordered list of timepoints TimePoints.getTimePointsOrdered().
    Author:
    Tobias Pietzsch <tobias.pietzsch@gmail.com>
