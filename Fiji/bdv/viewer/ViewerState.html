bdv.viewer

Interface ViewerState

  • All Known Implementing Classes:
    BasicViewerState, SynchronizedViewerState

    public interface ViewerState
    Reading and writing the BigDataViewer state:
    • interpolation and display mode
    • contained sources and source groups
    • activeness / currentness of sources and groups
    • current timepoint and transformation
    Author:
    Tobias Pietzsch

      • isSourceVisible

        boolean isSourceVisible(SourceAndConverter<?> source)
        Check whether the given source is visible.

        Whether a source is visible depends on the display mode:

        • In DisplayMode.SINGLE only the current source is visible.
        • In DisplayMode.GROUP the sources in the current group are visible.
        • In DisplayMode.FUSED all active sources are visible.
        • In DisplayMode.FUSEDROUP the sources in all active groups are visible.
        Returns:
        true, if source is visible
        Throws:
        NullPointerException - if source == null
        IllegalArgumentException - if source is not contained in the state (and not null).

      • isSourceVisibleAndPresent

        boolean isSourceVisibleAndPresent(SourceAndConverter<?> source)
        Check whether the given source is both visible and provides image data for the current timepoint.

        Whether a source is visible depends on the display mode:

        • In DisplayMode.SINGLE only the current source is visible.
        • In DisplayMode.GROUP the sources in the current group are visible.
        • In DisplayMode.FUSED all active sources are visible.
        • In DisplayMode.FUSEDROUP the sources in all active groups are visible.
        Additionally, the source must be present, i.e., provide image data for the current timepoint.
        Returns:
        true, if source is both visible and present
        Throws:
        NullPointerException - if source == null
        IllegalArgumentException - if source is not contained in the state (and not null).

      • getVisibleAndPresentSources

        Set<SourceAndConverter<?>> getVisibleAndPresentSources()
        Get the set of visible sources that also provide image data for the current timepoint.

        The returned Set is a copy. Changes to the set will not be reflected in the viewer state, and vice versa.

        Whether a source is visible depends on the display mode:

        • In DisplayMode.SINGLE only the current source is visible.
        • In DisplayMode.GROUP the sources in the current group are visible.
        • In DisplayMode.FUSED all active sources are visible.
        • In DisplayMode.FUSEDROUP the sources in all active groups are visible.
        Additionally, the source must be present, i.e., provide image data for the current timepoint.
        Returns:
        the set of sources that are both visible and present

      • removeSource

        boolean removeSource(SourceAndConverter<?> source)
        Remove source from the state (optional operation).

        Returns true, if source was removed from the state. Returns false, if source was not contained in state.

        The source is also removed from any groups that contained it. If source was current, then the first source in the list of sources is made current (if it exists).

        Returns:
        true, if list of sources changed as a result of the call
        Throws:
        NullPointerException - if source == null
        UnsupportedOperationException - if the operation is not supported by this ViewerState

      • removeSources

        boolean removeSources(Collection<? extends SourceAndConverter<?>> collection)
        Remove all sources in collection from the state (optional operation). Returns true, if at least one source was removed. Returns false, if none of the sources was present.

        Removed sources are also removed from any groups containing them. If the current source was removed, then the first source in the remaining list of sources is made current (if it exists).

        Returns:
        true, if list of sources changed as a result of the call.
        Throws:
        NullPointerException - if collection == null or any element of collection is null.
        UnsupportedOperationException - if the operation is not supported by this ViewerState
