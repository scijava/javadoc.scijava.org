Class ViewerState

  • java.lang.Object
    • bdv.viewer.state.ViewerState

  • public class ViewerState
extends Object
    Description of everything required to render the current image, such as the current timepoint, the visible and current sources and groups respectively, the viewer transformation, etc.

        Set the DisplayMode.
        • In single-source mode, only the current source (SPIM angle) is shown.
        • In fused mode, all active sources are blended.
