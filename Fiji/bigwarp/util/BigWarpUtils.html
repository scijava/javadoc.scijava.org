        public static void initTransform(ViewerPanel viewer)
        Set a "good" initial viewer transform. The viewer transform is chosen such that for the first source,
        • the XY plane is aligned with the screen plane,
        • the z = 0 slice is shown,
        • centered and scaled such that the full dim_x by dim_y is visible.
        This calls initTransform(int, int, boolean, ViewerState), using the size of the viewer's display component.
        Parameters:
        viewer - the viewer (containing at least one source) to have its transform set.
