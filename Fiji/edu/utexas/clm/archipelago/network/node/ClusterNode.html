        public void setIOStreams(InputStream is,
                         OutputStream os)
                  throws IOException,
                         TimeoutException,
                         InterruptedException
        Sets the InputStream and OutputStream used to communicate with the remote compute node. is and os are used to create a MessageXC.
        Parameters:
        is - InputStream to receive data from the remote machine
        os - OutputStream to send data to the remote machine
        Throws:
        IOException - if a problem arises opening one of the streams
        TimeoutException - if we time out while waiting for the remote node
        InterruptedException - if we are interrupted while waiting for the remote node

        public void addListener(NodeStateListener listener)
        Adds a NodeStateListener to the list of listeners that are notified when the state of this ClusterNode changes. Immediately upon addition, the listener is called with the current state, and given that the last state was ClusterNodeState.INACTIVE to indicate that this is the initial call.
        Parameters:
        listener - a NodeStateListener to register with the ClusterNode
