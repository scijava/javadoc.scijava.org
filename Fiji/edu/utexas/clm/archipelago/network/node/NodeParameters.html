      • merge

        public void merge(NodeParameters params)
        Sets default-empty values for parameters that are different between this NodeParameters and params. For instance, if params' user field is "John" and we have "Thomas" here, then this parameters' user will be set to "". On the other hand, if both are "Thomas," then the user string here will remain.
        Parameters:
        params - a NodeParameters to merge to this one
