        Start the ClusterNode given by the NodeParameters, by starting an ArchipelagoClient in a Fiji or ImageJ instance on the remote node. This method should handle the passed-in NodeShellListener, by passing it an InputStream and OutputStream corresponding to the remote node when they become ready, and calling execFinished when the remote connection (ie ssh shell) is closed.
