      • getDocumentationString

        public String getDocumentationString()
        Description copied from interface: ImgLibFunction
        Returns a documentation string that documents in enough details what the function does, and how. The String can use html syntax in it for formatting.

        Example:

        Gaussian convolution

        This function implements the isotropic gaussian convolution, as coded in ImgLib, effectively implementing a gaussian filter. Syntax:
        >> gauss(A, sigma)
        with A an image and sigma a number. Sigma is the standard deviation of the gaussian kernel applied to image A.
        Input image is converted to FloatType then convolved. If the source image is a 3D image, the convolution will be made in 3D as well.
