    public static class Logger.SlaveLogger
extends Logger
    A logger that wraps a master logger and uses it to echo any received message. This class is used to report progress of a sub-process. If it is sent to a subprocess, the master logger can show a progress in a range and from a starting point that can be specified.
    Author:
    Jean-Yves Tinevez - 2014
