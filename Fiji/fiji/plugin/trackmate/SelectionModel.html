      • selectTrack

        public void selectTrack(Collection<Spot> spots,
                        Collection<DefaultWeightedEdge> edges,
                        int direction)
        Search and add all spots and links belonging to the same track(s) that of given spots and edges to current selection. A direction parameter allow specifying whether we should include only parts upwards in time, downwards in time or all the way through.
        Parameters:
        spots - the spots to include in search
        edges - the edges to include in search
        direction - the direction to go when searching. Positive integers will result in searching upwards in time, negative integers downwards in time and 0 all the way through.
