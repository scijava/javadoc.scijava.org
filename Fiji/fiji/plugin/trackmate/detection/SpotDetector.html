Interface SpotDetector<T extends RealType<T> & NativeType<T>>

  • All Superinterfaces:
    Algorithm, Benchmark, OutputAlgorithm<List<Spot>>
    All Known Implementing Classes:
    BlockLogDetector, DogDetector, DownsampleLogDetector, LogDetector

    public interface SpotDetector<T extends RealType<T> & NativeType<T>>
extends OutputAlgorithm<List<Spot>>, Benchmark
    Interface for Spot detector classes, that are able to segment spots of a given estimated radius within a 2D or 3D image.

    Normally, concrete implementation are not expected to be multi-threaded. Indeed, the TrackMate trackmate generates one instance of the concrete implementation per thread, to process multiple frames simultaneously.
