    public class ModelFeatureUpdater
extends Object
implements ModelChangeListener, net.imglib2.algorithm.MultiThreaded
    A utility class that listens to the change occurring in a model, and updates its spot, edge and track features accordingly. Useful to keep the model in sync with manual editing.
