Interface EdgeAnalyzer

    public interface EdgeAnalyzer
extends net.imglib2.algorithm.Benchmark, FeatureAnalyzer, net.imglib2.algorithm.MultiThreaded
    Interface for analyzers that can compute scalar numerical features for an edge of a TrackMate model. An edge, or a link, is the single link that exists between two spots after tracking.

    • Method Detail

      • process

        void process(Collection<org.jgrapht.graph.DefaultWeightedEdge> edges,
             Model model)
        Scores a collection of link between two spots. The results must be stored in the FeatureModel.

        Note: ideally concrete implementation should work in a multi-threaded fashion for performance reason, when possible.

        Parameters:
        edges - the collection of edges whose features are to be calculated.
        model - the Model they belong to.

      • isLocal

        boolean isLocal()
        Returns true if this analyzer is a local analyzer. That is: a modification that affects only one edge requires the edge features to be re-calculated only for this edge. If false, any model modification involving an edge will trigger a recalculation over the whole track this edge belong to.

        Example of local edge feature: the edge length (distance between the two spots). This one does not depend on other edge values.

        Example of non-local edge feature: the local curvature of the trajectory, which depends on the neighbor edges.
