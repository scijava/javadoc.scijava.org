fiji.plugin.trackmate.features.spot

Interface SpotAnalyzerFactory<T extends RealType<T> & NativeType<T>>

  • All Superinterfaces:
    FeatureAnalyzer, SciJavaPlugin, TrackMateModule
    All Known Implementing Classes:
    ManualSpotColorAnalyzerFactory, SpotContrastAnalyzerFactory, SpotContrastAndSNRAnalyzerFactory, SpotIntensityAnalyzerFactory, SpotMorphologyAnalyzerFactory, SpotRadiusEstimatorFactory

    public interface SpotAnalyzerFactory<T extends RealType<T> & NativeType<T>>
extends FeatureAnalyzer
    Interface for factories that can generate a SpotAnalyzer configured to operate on a specific frame of a model.

    Concrete implementation should declare what features they can compute numerically, and make this info available in the SpotAnalyzerProvider that returns them.

    Feature key names are for historical reason all capitalized in an enum manner. For instance: POSITION_X, MAX_INTENSITY, etc... They must be suitable to be used as a attribute key in an xml file.

    Author:
    Jean-Yves Tinevez - 2012

        Returns a configured SpotAnalyzer ready to operate on the given frame (0-based) and given channel (0-based). The target frame image and the target spots are retrieved from the Model thanks to the given frame and channel index.
        Parameters:
        model - the Model to take the spots from.

        frame - the target frame to operate on.
