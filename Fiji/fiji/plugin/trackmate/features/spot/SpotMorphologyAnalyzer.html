    public class SpotMorphologyAnalyzer<T extends RealType<T>>
extends IndependentSpotFeatureAnalyzer<T>
    This SpotAnalyzer computes morphology features for the given spots.

    It estimates shape parameters by computing the most resembling ellipsoid from the pixels contained within the spot radius. From this ellipsoid, it determines what are its semi-axes lengths, and their orientation.

    In the 3D case, the features ELLIPSOIDFIT_SEMIAXISLENGTH_* contains the semi-axes lengths, ordered from the largest (A) to the smallest (C). ELLIPSOIDFIT_AXISPHI_* and ELLIPSOIDFIT_AXISTHETA_* give the orientation angles of the corresponding ellipsoid axis, in spherical coordinates. Angles are expressed in radians.

    • φ is the azimuth int the XY plane and its range is ]-π/2 ; π/2]
    • ϑ is the elevation with respect to the Z axis and ranges from 0 to π

    In the 2D case, ELLIPSOIDFIT_SEMIAXISLENGTH_A and ELLIPSOIDFIT_AXISPHI_A are always 0, the THETA angles are 0, and ELLIPSOIDFIT_AXISPHI_B and ELLIPSOIDFIT_AXISPHI_C differ by π/2.

    From the semi-axis length, a morphology index is computed. Spots are classified according to the shape of their most-resembling ellipsoid. We look for equality between semi-axes, with a certain tolerance, which value is SIGNIFICANCE_FACTOR.

    In the 2D case, if b > c are the semi-axes length

    • if b ≅ c, then this index has the value SpotMorphologyAnalyzerFactory.SPHERE
    • otherwise, it has the value SpotMorphologyAnalyzerFactory.PROLATE

    In the 2D case, if a > b > c are the semi-axes length

    • if a ≅ b ≅ c, then this index has the value SpotMorphologyAnalyzerFactory.SPHERE
    • if a ≅ b > c, then this index has the value SpotMorphologyAnalyzerFactory.OBLATE: the spot resembles a flat disk
    • if a > b ≅ c, then this index has the value SpotMorphologyAnalyzerFactory.PROLATE: the spot resembles a rugby ball
    • otherwise it has the value SpotMorphologyAnalyzerFactory.SCALENE; the spot's shape has nothing particular
