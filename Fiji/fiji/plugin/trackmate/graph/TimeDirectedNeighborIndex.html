      • predecessorListOf

        public List<Spot> predecessorListOf(Spot v)
        Returns the set of vertices which are the predecessors of a specified vertex. If the graph is a multigraph, vertices may appear more than once in the returned list. Because a list of predecessors can not be efficiently maintained, it is reconstructed on every invocation by duplicating entries in the neighbor set. It is thus more efficient to use predecessorsOf(Spot) unless duplicate neighbors are required.
        Parameters:
        v - the vertex whose predecessors are desired
        Returns:
        all predecessors of the specified vertex
