      • setGUIStateString

        public void setGUIStateString(String stateKey)
        Sets the GUI current state via a key string. Registered descriptors are parsed until one is found that has a matching key ( WizardPanelDescriptor.getKey()). Then it is displayed. If a matching key is not found, nothing is done, and an error is logged in the LogPanel.

        This method is typically called to restore a saved GUI state.

        Parameters:
        stateKey - the target state string.
