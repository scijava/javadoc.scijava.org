fiji.plugin.trackmate.io

Class IcyXmlReader

  • java.lang.Object
    • fiji.plugin.trackmate.io.IcyXmlReader

  • public class IcyXmlReader
extends Object
    A reader for the XML files generated in the TrackManager of the Icy software Icy/Track Manager .

    The reader requires a calibration s to operate, so as to generate a proper model objects. Indeed, the Icy file format stores the detection coordinates in pixel coordinates, while TrackMate uses image coordinates. We need a proper calibration for conversion.

        Creates a new ICY importer from parsing the specified XML file, with default spot radius, spot quality and edge weight values.

        If an error triggered upon parsing the file, the flag returned by isReadingOk() will return false. The content of the error message can then be obtained through getErrorMessage().

        Creates a new ICY importer from parsing the specified XML file.

        If an error triggered upon parsing the file, the flag returned by isReadingOk() will return false. The content of the error message can then be obtained through getErrorMessage().

        Parameters:
        file - the ICY tracks XML file. Will generate an error if the XML file is not a proper ICY tracks XML file.
        calibration - the spatial calibration as a double[] array. Must store the pixel sizes dx, dy, dz.
        dt - the frame interval in time units.
        radius - the radius, in image coordinates, for the spot created upon loading.
        spotQuality - the quality for spot created upon loading.
        edgeWeight - the weight for edges created upon loading.
