      • getLowerRight

        protected Jama.Matrix getLowerRight(Jama.Matrix topLeft,
                                    double cutoff)
        Takes the submatrix of costs defined by rows 0 to numRows - 1 and columns 0 to numCols - 1, transpose it, and sets any non-BLOCKED value to be cutoff.

        The reasoning for this is explained in the supplementary notes of the paper, but basically it has to be made this way so that the LAP is solvable.

      • getAlternativeScores

        protected Jama.Matrix getAlternativeScores(int n,
                                           double cutoff)
        Sets alternative scores in a new matrix along a diagonal. The new matrix is n x n, and is set to BLOCKED everywhere except along the diagonal that runs from top left to bottom right.
