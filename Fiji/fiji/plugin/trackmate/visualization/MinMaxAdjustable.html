      • setAutoMinMaxMode

        void setAutoMinMaxMode(boolean autoMode)
        Sets the behavior of this object regarding the min & max calculation.

        If the flag passed is true, then the min & max will be recalculated by the instance everytime it is meaningful to do so, by calling the autoMinMax() method.

        If false, the min & max will not be recalculated. They will keep their current value unless setMinMax(double, double) is called.

        Parameters:
        autoMode - the behavior flag.

      • isAutoMinMaxMode

        boolean isAutoMinMaxMode()
        Returns whether the automatic scaling mode is activated or not for this object.
        Returns:
        true if the automatic scaling mode is on.

      • setFrom

        void setFrom(MinMaxAdjustable minMaxAdjustable)
        Copies the min, max and auto fields from the specified MinMaxAdjustable to this object.

        Not that if isAutoMinMaxMode() is set to true, the min and max value will not be copied over.
