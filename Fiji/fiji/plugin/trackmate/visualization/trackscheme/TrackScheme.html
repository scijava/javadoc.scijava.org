      • modelChanged

        public void modelChanged(ModelChangeEvent event)
        Used to catch spot creation events that occurred elsewhere, for instance by manual editing in the AbstractTrackMateModelView.

        We have to deal with the graph modification ourselves here, because the Model model holds a non-listenable JGraphT instance. A modification made to the model would not be reflected on the graph here.

        Parameters:
        event - the ModelChangeEvent.
