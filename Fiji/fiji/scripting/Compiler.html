      • getClasspath

        public static String getClasspath()
        Modified from Johannes Schindelin's Refresh_Javas.getPluginsClassPath method. Obtain the complete classpath for Fiji or ImageJ, which includes the basic java classpath plus, in addition, the jar files for all plugins and for all included libraries in the /jars folder. When called from outside ImageJ or Fiji, it returns the basic java classpath.

        public static Compiler.Result compile(String path,
                                      String classPath,
                                      String rootDir,
                                      String outPath,
                                      String[] extraArgs)
                               throws ClassNotFoundException,
                                      NoSuchMethodException,
                                      IllegalAccessException,
                                      InvocationTargetException
        Borrowing very heavily from Johannes Schindelin's Refresh_Javas.compile method.
        Parameters:
        path - The file path to the .java file to compile.
        classPath - The java classpath, which for Fiji includes all .jar files under /jars and /plugins and can be obtained from getClasspath().
        rootDir - Optional (can be null). The top-level root directory from which the java file is to be compiled.
        outPath - Optional (can be null), the target file path for the .class file(s). Will use the same folder as the .java file when null.
        extraArgs - Optional (can be null). Additional arguments for the javac compiler.
        Returns:
        A new Compiler.Result object indicating success or not, and if so an error message.
        Throws:
        ClassNotFoundException
        NoSuchMethodException
        IllegalAccessException
        InvocationTargetException
