Class NumberParser

  • java.lang.Object
    • fiji.util.NumberParser

  • public final class NumberParser
extends Object
    A static utility class for parsing numbers from strings. This class differs from Integer.parseInt(String), Double.parseDouble(String), etc., in that it respects the current Locale, allowing to parse numbers in various formats worldwide.
    Author:
    Curtis Rueden

      • parseNumber

        public static Number parseNumber(String number)
        Parses a Number from the given string, respecting the default Locale.

        Strings representing integers or fractions with a decimal point, optionally signed, are parsed with the US locale. Other strings are first parsed with the default locale and if that fails, parsing is re-attempted using the en-US locale as well before giving up.

        Throws:
        NumberFormatException - if the number cannot be parsed

      • parseInteger

        public static int parseInteger(String number)
        Parses an int from the given string, respecting the default Locale. If a decimal number is given as input, the result is rounded to the nearest integer; see NumberFormat.getIntegerInstance() for details.

        If parsing with the default locale fails, parsing is reattempted using the en-US locale as well before giving up.

        Throws:
        NumberFormatException - if the number cannot be parsed
