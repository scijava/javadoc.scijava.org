hr.irb.fastRandomForest

Class DataCache

  • java.lang.Object
    • hr.irb.fastRandomForest.DataCache

  • public class DataCache
extends Object
    Stores a dataset that in FastRandomTrees use for training. The data points are stored in a single-precision array indexed by attribute first, and then by instance, to make access by FastRandomTrees faster. Also stores the sorted order of the instances by any attribute, can create bootstrap samples, and seed a random number generator from the stored data.
    Author:
    Fran Supek (fran.supek[AT]irb.hr)
