Class Volume

  • java.lang.Object
    • ij3d.Volume
  • Direct Known Subclasses:
    AreaListVolume, ImgLibVolume, VoltexVolume

    public class Volume
extends Object
    This class encapsulates an image stack and provides various methods for retrieving data. It is possible to control the loaded color channels of RGB images, and to specify whether or not to average several channels (and merge them in this way into one byte per pixel). Depending on these settings, and on the type of image given at construction time, the returned data type is one of INT_DATA or BYTE_DATA.
    Author:
    Benjamin Schmid
