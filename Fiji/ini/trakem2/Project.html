      • newFSProject

        public static Project newFSProject(String arg,
                                   TemplateThing template_root,
                                   String storage_folder)
        Creates a new project to be based on .xml and image files, not a database. Images are left where they are, keeping the path to them. If the arg equals 'blank', then no template is asked for; if template_root is not null that is used; else, a template file is asked for.
        Parameters:
        arg - Either "blank", "amira", "stack" or null. "blank" will generate a default template tree; "amira" will ask for importing an Amira file; "stack" will ask for importing an image stack (single multi-image file, like multi-TIFF).
        template_root - May be null, in which case a template DTD or XML file will be asked for, unless arg equals "blank".
        storage_folder - If null, a dialog asks for it.
