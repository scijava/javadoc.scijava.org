jRenderer3D

Class Line3D

  • java.lang.Object
    • jRenderer3D.Line3D

  • public class Line3D
extends Object
    This class represents a line in 3D coordinate system. It is defined by its start point (x1, y1, z1) and its end point (x2, y2, z2), and its color.

    At the moment it is not possible to set line width.

    Author:
    Kai Uwe Barthel
