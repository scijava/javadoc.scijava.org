      • getRegionalExtremaCenters

        public ArrayList<float[]> getRegionalExtremaCenters()
        Description copied from interface: RegionalExtremaFinder
        Computes the average coordinate of the extrema locations. This average coordinates are returned, and represents the "center" of the regional maximum. Note that this is not guaranteed to be in the regional maximum itself (imagine a regional maximum that is a ring shape; the direct center is not part of the regional maximum itself, but is what would be computed by this function). Return null if the Algorithm.process() method has not bee called.
        Specified by:
        getRegionalExtremaCenters in interface RegionalExtremaFinder<T extends RealType<T>>
        Returns:
        The coordinates of the "center pixel" of the regional maximum. It will always be a 3-elements float arrays, even for 2D case.
