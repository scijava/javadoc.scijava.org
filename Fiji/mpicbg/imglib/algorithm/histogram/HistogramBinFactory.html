Interface HistogramBinFactory<T extends Type<T>>

  • Type Parameters:
    T - The type of Type that this HistogramBinFactory pertains to.
    All Known Implementing Classes:
    DiscreteIntHistogramBinFactory, RealHistogramBinFactory

    public interface HistogramBinFactory<T extends Type<T>>
    The HistogramBinFactory interface is used by Histogram to generate HistogramBins and HistogramKeys. In addition, a HistogramBinFactory implements a method used to determine if two Type's are equivalent to each other, much as in .equals().
    Author:
    Larry Lindsey
