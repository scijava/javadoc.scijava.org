        public HistogramKey(int hc,

        Create a HistogramKey with hash code hc, representative t, and HistogramBinFactory factory.
        Parameters:
        hc - the hash code to be returned by this HistogramKey's hashCode().
        t - the representative Type. Preferably, this should be the center Type of the HistogramBin to be keyed, but it only need be one that would be binned into that bin.
        factory - the HistogramBinFactory corresponding to this key. Typically, this should be the HistogramBinFactory that generated this key.
