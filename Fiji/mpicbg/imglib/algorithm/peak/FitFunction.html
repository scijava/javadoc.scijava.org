Interface FitFunction

  • All Known Implementing Classes:
    GaussianMultiDLM

    public interface FitFunction
    Interface for fitting-target functions. <> Functions defined here are scalar, multi-dimensional ones. We impose that they can return a gradient, so as to use various curve-fitting scheme.
