    public static interface StatisticalOperation.StatisticsCollectionStrategy<R extends RealType<R>>
    Implements a strategy for populating the sorted list associated with this StatisticalOperation. This could mean either re-populating and resorting this list on every iteration, which might be reasonably efficient for small structuring elements, or it could mean being smart about the boundaries, since we expect our cursor to move maybe only one position for every iteration.
