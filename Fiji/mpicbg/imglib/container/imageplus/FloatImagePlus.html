      • FloatImagePlus

        public FloatImagePlus(ImagePlusContainerFactory factory,
                      int[] dim,
                      int entitiesPerPixel)
        Standard constructor as called by default factories. Note that this constructor does not know about the meaning of dimensions > 1, and will use them in the ImagePlus default order x,y,c,z,t. That is, from two dimensions, it will create an x,y image, from three dimensions, an x,y,c image, and from four dimensions, an x,y,c,z image.
