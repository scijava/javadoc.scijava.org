    public abstract class AbstractSpecialCursor<T extends Type<T>>
extends Object
implements LocalizableCursor<T>
    This abstract cursor offer facilities for specialized cursor that are based on a LocalizableByDimCursor whose iteration domain is imposed. This abstract class itself is not really interesting, see sub-classes.

      • getNPixels

        public abstract int getNPixels()
        Return the number of pixels this cursor will iterate on (or, the number of iterations it will do before exhausting). This is useful when one needs to know the number of pixel iterated on in advance. For instance: 
         DiscCursor dc = new DiscCursor(img, center, 5);
 int arrraySize = sc.getNPixels();
 float[] pixelVal = new float[arraySize];
 int index = 0;
 while (sc.hasNext()) {
        sc.fwd();
        pixelVal[index] = sc.getType().getRealFloat();
        index++;
 }
