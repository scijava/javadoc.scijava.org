    public abstract class DomainCursor<T extends Type<T>>
extends CoordsCursor<T>
    Abstract class for CoordsCursors that iterate over a domain which size can be determined by a single parameter. For instance: cubes, spheres, circles.

    This abstract layer add a single method: setSize(float) that reconfigures the domain to iterate on.
