Class HyperSphereIterator<T extends Type<T>>

  • java.lang.Object
    • mpicbg.imglib.cursor.special.HyperSphereIterator<T>
  • Type Parameters:

    All Implemented Interfaces:
    Iterator<T>, Cursor<T>, Iterable, Localizable, LocalizableCursor<T>, Dimensionality

    public class HyperSphereIterator<T extends Type<T>>
extends Object
implements Iterable, LocalizableCursor<T>, Localizable
    Iterate over all pixels in an n-dimensional sphere. This implementation is Container agnostic and thus the general purpose version of the Iterator. For specific Containers, a different implementation might be more efficient.
