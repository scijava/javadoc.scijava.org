      • computePDRecursive

        protected static int[][] computePDRecursive(int tolerance,
                                            int n,
                                            int offset)
        Computes recursively how to create different PointDescriptors of the same Point when some tolerance is allowed, e.g. there are outliers which should be identified. For 3 neighbors allowing 2 outliers and starting with the second nearest neigbor (the nearest one is in most cases the one you use as input) the result looks like that: (Each coloum will be one PointDescriptor) |1| |1| |1| |2| |1| |1| |2| |1| |2| |3| |2| |2| |3| |3| |2| |3| |3| |4| |4| |4| |3| |4| |4| |4| |5| |5| |5| |5| |5| |5|
        Parameters:
        tolerance - - How many tolerance is accepted [0 ... m]
        n - - initialized with the number of neighbors
        offset - - the starting position for the neighbors, usually 1
        Returns:
        an array containing the neighbors for each PointDescriptor as array
