mpicbg.spim.data

Class SpimDataExample2

  • java.lang.Object
    • mpicbg.spim.data.SpimDataExample2

  • public class SpimDataExample2
extends Object
    The methods exampleViewSetups(), exampleTimePoints(), exampleViewRegistrations(), and exampleSequenceDescription() construct ViewSetups, TimePoints, ViewRegistrations, and SequenceDescription respectively. These are serialized to xml, deserialized, and serialized again. The two serialized XML versions are printed (and should be the same).

    Tobias Pietzsch <tobias.pietzsch@gmail.com>
