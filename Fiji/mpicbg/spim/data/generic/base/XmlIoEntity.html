Class XmlIoEntity<T extends Entity>

  • java.lang.Object
    • mpicbg.spim.data.generic.base.XmlIoEntity<T>
  • Type Parameters:
    T - the entity type
    Direct Known Subclasses:
    XmlIoBasicViewSetup, XmlIoNamedEntity

    public class XmlIoEntity<T extends Entity>
extends Object
    Writes and reads objects of a specific subclass of Entity as XML elements. When reading/writing an object, XmlIoEntity takes care of the id of the object. Other attributes are handled in derived classes. Derived classes should add the tag names of children that they handle to the set handledTags. When reading an object from an XML element, unhandled children of the element are stored as associated with the objects id. When writing an object with the same id to an XML element these children will be added. This provides a way to read / modify / write spimdata XML files without knowing the details of possible extensions.

    Tobias Pietzsch <tobias.pietzsch@gmail.com>
