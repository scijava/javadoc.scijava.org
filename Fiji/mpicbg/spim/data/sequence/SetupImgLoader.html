Interface SetupImgLoader<T>

    public interface SetupImgLoader<T>
extends BasicSetupImgLoader<T>
    A BasicSetupImgLoader that is able to also provide each image converted to FloatType. Moreover, it provides voxel size and image size for each image (usually without needing to completely read the image).
    Author:
    Tobias Pietzsch <tobias.pietzsch@gmail.com>
