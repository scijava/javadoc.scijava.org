Class ViewSetup

  • All Implemented Interfaces:
    Comparable<ViewSetup>

    public class ViewSetup
extends BasicViewSetup
implements Comparable<ViewSetup>
    A collection of parameters describing the setup for a particular stack coming from a SPIM microscope (angle, illumination direction, etc). A ViewSetup is a BasicViewSetup that must have a channel, angle, and illumination direction.
    Author:
    Tobias Pietzsch <tobias.pietzsch@gmail.com>, Stephan Preibisch (stephan.preibisch@gmx.de)
