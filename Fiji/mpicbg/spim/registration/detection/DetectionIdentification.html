mpicbg.spim.registration.detection

Class DetectionIdentification<S extends DetectionIdentification<S,T>,T extends DetectionView<S,T>>

  • java.lang.Object
    • mpicbg.spim.registration.detection.DetectionIdentification<S,T>
  • Direct Known Subclasses:
    BeadIdentification, NucleusIdentification

    public abstract class DetectionIdentification<S extends DetectionIdentification<S,T>,T extends DetectionView<S,T>>
extends Object
    The DetectionIdentification object stores the link (via ID) to an Detection-object and not an actual instance, but a link to the ViewDataBeads object where it belongs to. This is necessary for storing/loading AbstractDetectionon-relationships to/from a text file. The AbstractDetection-objects of every ViewDataBeads have an ArrayList of DetectionIdentification objects telling which other AbstractDetections are correspondence candidates or true correspondences.

    Stephan Preibisch
