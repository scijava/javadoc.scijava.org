mpicbg.stitching

Class PairWiseStitchingImgLib

  • java.lang.Object
    • mpicbg.stitching.PairWiseStitchingImgLib

  • public class PairWiseStitchingImgLib
extends Object
    Pairwise Stitching of two ImagePlus using ImgLib1 and PhaseCorrelation. It deals with aligning two slices (2d) or stacks (3d) having an arbitrary amount of channels. If the ImagePlus contains several time-points it will only consider the first time-point as this requires global optimization of many independent 2d/3d <-> 2d/3d alignments.
    Author:
    Stephan Preibisch (stephan.preibisch@gmx.de)
