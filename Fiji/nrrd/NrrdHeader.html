nrrd

Class NrrdHeader

  • java.lang.Object
    • nrrd.NrrdHeader

  • public class NrrdHeader
extends Object
    Class to read and represent the header of a NRRD file.

    This is for a generic nrrd reader. The only special processing that it does is:

    • change field names to lower case
    • turn nrrd field names into a standard form (e.g. "axismin"->"axis min")
    • doesn't attempt to white space separate content field
    • records content-type tag if present
    Author:
    Greg Jefferis
