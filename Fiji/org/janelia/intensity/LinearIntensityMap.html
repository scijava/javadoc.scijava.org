org.janelia.intensity

Class LinearIntensityMap<T extends net.imglib2.type.numeric.RealType<T>>

  • java.lang.Object
    • org.janelia.intensity.LinearIntensityMap<T>

  • public class LinearIntensityMap<T extends net.imglib2.type.numeric.RealType<T>>
extends Object
    Transfer intensities by a linear function y=ax+b with the coefficients a and b being stored as a RealComposite<T> 2d-vector (a, b) in a map. The map is a RealRandomAccessible and the defined interval as passed as an independent parameter. If the coefficients are passed as a raster, the interval is that of a raster. The map is scaled such that it applies to the interval of the input image.
    Author:
    Stephan Saalfeld saalfelds@janelia.hhmi.org
