Class Orthoslice

  • java.lang.Object
    • voltex.VolumeRenderer
      • orthoslice.Orthoslice
  • All Implemented Interfaces:

    public class Orthoslice
extends VolumeRenderer
    Orthoslice extends VolumeRenderer and modifies it in a way so that not the whole volume is displayed, but only three orthogonal slices through the volume.
    Author:
    Benjamin Schmid

      • Orthoslice

        public Orthoslice(ij.ImagePlus img,
                  org.scijava.vecmath.Color3f color,
                  float tr,
                  boolean[] channels)
        Initializes a new Orthoslice with the given image, color, transparency and channels. By default, the slice indices go through the center of the image stack.
        Parameters:
        img - The image stack
        color - The color this Orthoslice should use
        tr - The transparency of this Orthoslice
        channels - A boolean[] array which indicates which color channels to use (only affects RGB images). The length of the array must be 3.
