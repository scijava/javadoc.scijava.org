  • public class InputCheck<T extends net.imglib2.type.numeric.RealType<T>>
extends Algorithm<T>
    This class implements some basic checks for the input image data. For instance: Is the percentage of zero-zero or saturated pixels too high? Also, we get basic image properties / stats from imglib2, and also the colocalization job name from the DataContainer and allow implementations of ResultHandler to report them.
