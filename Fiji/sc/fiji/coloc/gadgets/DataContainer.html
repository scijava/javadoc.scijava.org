Class DataContainer<T extends net.imglib2.type.numeric.RealType<T>>

    T - 

    public class DataContainer<T extends net.imglib2.type.numeric.RealType<T>>
extends Object
    The DataContainer keeps all the source data, jobName, pre-processing results and algorithm results that have been computed. It allows a ResultsHandler implementation to get most of its content and makes the source image and channel information available to a client.

        Creates a new DataContainer for a specific set of image and channel combination. It will give access to the image according to the mask passed. It is expected that the mask is of the same size as an image slice. Default thresholds, min, max and mean will be set according to the mask as well.
        Parameters:
        src1 - The channel one image source
        src2 - The channel two image source
        ch1 - The channel one image channel
        ch2 - The channel two image channel
        mask - The mask to use
        offset - The offset of the ROI in each dimension
        size - The size of the ROI in each dimension
        Throws:
        MissingPreconditionException
