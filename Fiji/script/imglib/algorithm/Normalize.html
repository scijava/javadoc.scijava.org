    public class Normalize<N extends NumericType<N>>
extends AbstractNormalize<N>
    Becomes a normalized version of the given image, within min and max bounds, where all pixels take values between 0 and 1. The constructor accepts any of IFunction, ColorFunction, Image. Images may be of any RealType or RGBALegacyType. In the latter case, each color channel is normalized independently. When the min equals the max, the result is an image with zero values.
