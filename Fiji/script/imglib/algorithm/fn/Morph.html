    public abstract class Morph<T extends RealType<T>>
extends Image<T>
    Morphological operations such as Open, Close, Erode and Dilate. The operation takes any of CUBE, BALL, BAR, each with a length. In ImageJ, the equivalent would be a CUBE with length 3. The lengthDim is only useful for BAR, and specifies in which axis to grow the bar.
