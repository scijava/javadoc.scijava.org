    public class DoGPeaks<N extends RealType<N>>
extends ArrayList<float[]>
    Perform a difference of Gaussian on the given Image, and this class itself becomes the List of found peaks, each as a float[] array that specifies its position. See also DifferenceOfGaussian, DifferenceOfGaussianPeak.
