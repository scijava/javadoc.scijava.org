Package sholl.gui

Implements GUI customizations required by the Sholl Analysis plugin.

See: Description

  • Class Summary 
    Class Description
    EnhancedGenericDialog
    Enhances GenericDialog with a few additional features, including scrollbars as soon as the Dialog is too large to be displayed, ability to use the "help" button to display a drop-down menu, and labels featuring clickable hyperlinks.
    EnhancedResultsTable
    Slight improvements to ResultsTable
    EnhancedWaitForUserDialog
    Enhances WaitForUserDialog with a few additional features.
    Utils
    Provides customizations for ImageJ dialogs when not running headless.
