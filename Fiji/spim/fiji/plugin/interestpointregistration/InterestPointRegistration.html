        public InterestPointRegistration(SpimData2 spimData,
                                 List<ViewId> viewIdsToProcess,
                                 List<ChannelProcess> channelsToProcess)
        Instantiate the interest point registration. It is performed for a spimdata object on a subset of angles, channels, illuminations and timepoints. Each channel is linked to a certain type of detections (e.g. beads, nuclei), hence the ChannelProcess object.
        Parameters:
        spimData -
        viewIdsToProcess - - which view id's to register
        channelsToProcess - - which Channel uses which label for registration

      • register

        public boolean register(GlobalOptimizationType registrationType,
                        boolean save,
                        boolean collectStatistics)
        Registers all timepoints. No matter which matching is done it is always the same principle. First all pairwise correspondences are established, and then a global optimization is computed. The global optimization can is done in subsets, where the number of subsets >= 1.
        Parameters:
        registrationType - - which kind of registration
        save - - if you want to save the correspondence files
        Returns:
