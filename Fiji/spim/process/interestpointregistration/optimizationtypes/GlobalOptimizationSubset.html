spim.process.interestpointregistration.optimizationtypes

Class GlobalOptimizationSubset

  • java.lang.Object
    • spim.process.interestpointregistration.optimizationtypes.GlobalOptimizationSubset

  • public class GlobalOptimizationSubset
extends Object
    Defines a subset of views that need to be matched and then a global optimization be computed on. In case of a registration of individual timepoints, each timepoint would be one GlobalOptimizationSubset, for an all-to-all matching there is only one object.
    Author:
    Stephan Preibisch (stephan.preibisch@gmx.de)
