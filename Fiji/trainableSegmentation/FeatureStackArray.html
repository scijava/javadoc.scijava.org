Class FeatureStackArray

  • java.lang.Object
    • trainableSegmentation.FeatureStackArray

  • public class FeatureStackArray
extends Object
    This class stores the feature stacks of a set of input slices. It can be used so for 2D stacks or as the container of 3D features (by using a feature stack per section).
    Author:
    Ignacio Arganda-Carreras (iarganda@mit.edu)
