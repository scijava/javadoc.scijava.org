  • public class AdjustedRandError
extends Metrics
    This class implements the adjusted Rand error, defined as the 1 - adjusted Rand index. We follow the Rand index definition described by Lawrence Hubert and Phipps Arabie \cite{Hubert85}. BibTeX:
