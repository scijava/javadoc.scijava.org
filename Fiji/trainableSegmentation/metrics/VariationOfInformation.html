trainableSegmentation.metrics

Class VariationOfInformation

  • java.lang.Object
    • trainableSegmentation.metrics.Metrics
      • trainableSegmentation.metrics.VariationOfInformation

  • public class VariationOfInformation
extends Metrics
    This class implements the variation of information metric, used to compare clusters or segmentations. Two implementations are given, the classic one, and a foreground-restricted version that prunes out the background pixels of the ground truth during the metric calculation.
