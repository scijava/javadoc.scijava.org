Class VolumeRenderer

  • java.lang.Object
    • voltex.VolumeRenderer
  • All Implemented Interfaces:
    AxisConstants
    Direct Known Subclasses:
    MultiOrthoslice, Orthoslice

    public class VolumeRenderer
extends Object
implements AxisConstants
    This class is the main class for creating a volume rendering. It consists of a root branch group, which holds a Switch of OrderedGroups. Each OrderedGroup holds the shapes for one direction. One shape consists of a rectangular geometry on which a texture from the given image data is mapped.
    Author:
    Benjamin Schmid

      • VolumeRenderer

        public VolumeRenderer(ij.ImagePlus img,
                      Color3f color,
                      float tr,
                      boolean[] channels)
        Constructor. Initializes this VolumeRenderer with the given image, color, transparency and channels. This does not yet start the rendering. To do so, fullReload() needs to be called.
        Parameters:
        img - the image stack to be rendered.
        color - the color in which this rendering should be displayed
        tr - the transparency value for this volume rendering
        channels - the rgb channels which should be used. This must be a boolean[] array of length three, one for red, green and blue respectively.
