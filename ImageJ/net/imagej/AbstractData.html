        public void incrementReferences()
        Description copied from interface: Data
        Adds to the data object's reference count. Typically this is called when the data object is added to a display. Implementers of this interface may want to emit Events when the reference count goes to 1. AbstractData does exactly this to let interested parties know that a data object has come into use.

        public void decrementReferences()
        Description copied from interface: Data

Implementers of this interface may want to emit Events when the reference count goes to 0. AbstractData does exactly this to let interested parties know that the data object is no longer in use.
