net.imagej.axis

Interface Axis

  • All Known Subinterfaces:
    CalibratedAxis, LinearAxis, TypedAxis
    All Known Implementing Classes:
    AbstractCalibratedAxis, ChapmanRichardsAxis, DefaultLinearAxis, DefaultTypedAxis, ExponentialAxis, ExponentialRecoveryAxis, GammaVariateAxis, GaussianAxis, IdentityAxis, InverseRodbardAxis, LogLinearAxis, PolynomialAxis, PowerAxis, RodbardAxis, Variable1Axis, Variable2Axis, Variable3Axis, Variable4Axis, VariableAxis

    public interface Axis
    A dimensional axis of an AnnotatedSpace.

    This interface does not define any explicit API, but serves as an extension point for annotated EuclideanSpaces: subtypes of Axis can be defined to include concepts such as axis names, labels and calibrations, and then assigned to the axes of an AnnotatedSpace.

    Author:
    Curtis Rueden
