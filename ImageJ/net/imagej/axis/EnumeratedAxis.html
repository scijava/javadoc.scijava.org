    public class EnumeratedAxis
extends AbstractCalibratedAxis
    A CalibratedAxis whose coordinate values are explicitly enumerated by a list. Out-of-bounds calibrated values are linearly extrapolated from the two nearest known coordinates. Non-integer calibrated values are linearly interpolated from the two nearest integer indices.
    Author:
    Curtis Rueden

    • Constructor Detail

      • EnumeratedAxis

        public EnumeratedAxis(AxisType type,
                      List<? extends Number> values)
        Creates an axis whose calibrated values are defined by the given list of values. This is a convenience constructor which copies the list values to a new array internally.
        Parameters:
        type - The type of axis (e.g. Axes.X or Axes.TIME.
        values - List of calibrated values. The first element of the list is the calibrated value at raw position 0, the second element is the calibrated value at raw position 1, and so forth. Intermediate and out-of-bounds calibrated values are inferred by linear interpolation or extrapolation, respectively.
