Interface DatasetView

  • All Superinterfaces:
    Contextual, DataView, EuclideanSpace, ImageJPlugin, Localizable, Positionable, PositionableByAxis, RealLocalizable, SciJavaPlugin
    All Known Implementing Classes:
    DefaultDatasetView

    public interface DatasetView
extends DataView
    A linkage between a Dataset and a Display. The view takes care of mapping the N-dimensional data into a representation suitable for showing onscreen.
    Author:
    Curtis Rueden, Barry DeZonia

      • setChannelRange

        void setChannelRange(int c,
                     double min,
                     double max)
        Sets the minimum and maximum values of the display range, for the given channel.

        NB: This is a different range than that set by ImageMetadata.setChannelMinimum(int, double) and ImageMetadata.setChannelMaximum(int, double); the latter methods set the minimum and maximum data values for that channel, independent of any visualization. They are typically kept synced with the actual data via code such as net.imglib2.algorithm.stats.ComputeMinMax.

      • setChannelRanges

        void setChannelRanges(double min,
                      double max)
        Sets the minimum and maximum values of the display range, globally for all channels.

        NB: This is a different range than that set by ImageMetadata.setChannelMinimum(int, double) and ImageMetadata.setChannelMaximum(int, double); the latter methods set the minimum and maximum data values for that channel, independent of any visualization. They are typically kept synced with the actual data via code such as net.imglib2.algorithm.stats.ComputeMinMax.

      • autoscale

        void autoscale(int c)
        Autoscales the display range to match the data range, for the given channel.

        NB: The data range is obtained first from ImageMetadata.getChannelMinimum(int) and ImageMetadata.getChannelMaximum(int); if they are not set there, they are computed and cached for later use.
