net.imagej.display

Interface ImageCanvas

  • All Superinterfaces:
    Pannable, Zoomable
    All Known Implementing Classes:
    DefaultImageCanvas

    public interface ImageCanvas
extends Pannable, Zoomable
    A canvas upon which an ImageDisplay draws its output.

    An image canvas supports two different coordinate systems, and provides a mechanism for conversion between them. The first is panel coordinates, which are in pixels onscreen. The second is data coordinates, which match the linked ImageDisplay's original coordinate system.

    Author:
    Grant Harris, Curtis Rueden
