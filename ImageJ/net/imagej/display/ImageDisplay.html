    DefaultImageDisplay

    public interface ImageDisplay
extends Display<DataView>, CalibratedRealInterval<CalibratedAxis>, PositionableByAxis, Interval
    An image display is a Display for visualizing Data objects. It operates directly on DataViews, which wrap the Data objects and provide display settings for the data.
