    public class DatasetRestructuredEvent
extends DataRestructuredEvent
    An event indicating a Dataset's dimensional extents may have changed. This occurs, for example, if the dataset's linked ImgPlus changes.

    This event is typically fired as part of a call to Dataset.rebuild().
