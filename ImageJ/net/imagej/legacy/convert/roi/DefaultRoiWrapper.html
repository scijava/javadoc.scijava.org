net.imagej.legacy.convert.roi

Class DefaultRoiWrapper<R extends Roi>

  • java.lang.Object
    • net.imagej.legacy.convert.roi.DefaultRoiWrapper<R>
  • All Implemented Interfaces:
    Predicate<Localizable>, IJRoiWrapper<R,Localizable>, Dimensions, EuclideanSpace, Interval, RealInterval, Mask, MaskInterval, MaskPredicate<Localizable>

    public class DefaultRoiWrapper<R extends Roi>
extends Object
implements IJRoiWrapper<R,Localizable>, MaskInterval
    Wraps any Roi as a MaskInterval. The test(...) method of this class, simply calls contains(...) on the underlying ImageJ 1.x Roi. This is intended to wrap existing ImageJ 1.x Rois which do not translate well to existing Imglib2 interfaces.
    Author:
    Alison Walter
