    public class DefaultRoiWrapper<R extends ij.gui.Roi>
extends Object
implements IJRoiWrapper<R,net.imglib2.Localizable>, net.imglib2.roi.MaskInterval
    Wraps any Roi as a MaskInterval. The test(...) method of this class, simply calls contains(...) on the underlying ImageJ 1.x Roi. This is intended to wrap existing ImageJ 1.x Rois which do not translate well to existing Imglib2 interfaces.

    Alison Walter
