    public class IrregularPolylineRoiWrapper
extends Object
implements IJRealRoiWrapper<ij.gui.PolygonRoi>
    Wraps a PolygonRoi of type Roi.POLYLINE or Roi.FREELINE with a non zero width as a RealMaskRealInterval. This can also be used to wrap Roi.POLYLINE which have been spline fitted.

    This is only intended to wrap existing ImageJ 1.x Rois, not to create ImgLib2 Rois backed by ImageJ 1.x Rois.

    Alison Walter
