    public class LegacyImageDisplayService
extends org.scijava.service.AbstractService
implements ImageDisplayService
    HACK - this whole service exists as a hack to facilitate running commands that require a Dataset without requiring ImageJ2 synchronization to be enabled via Edit>Options>ImageJ2.

    NB: DisplayService and ObjectService will NOT be synchronized with the state represented by this service. Use cases querying these services require ImageJ2 synchronization to be enabled.

    This service will be made obsolete by creating lightweight wrappers.

    Mark Hiner
