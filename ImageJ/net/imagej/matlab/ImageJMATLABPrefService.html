    public class ImageJMATLABPrefService
extends DefaultPrefService
    A PrefService implementation specific for use within MATLAB. MATLAB has known bugs regarding java.util.prefs.Preferences, so we need the ability to disable their use, via a setEnabled(boolean) method.

    Mark Hiner

      • setEnabled

        public static void setEnabled(boolean enabled)
        Sets whether or not this PrefService is enabled. If so, it will behave like a DefaultPrefService. If not, all methods will return empty objects, 0 values, or be NO-OPs.

        NB: this method needs to be static so that it can be called before a Context is created.
