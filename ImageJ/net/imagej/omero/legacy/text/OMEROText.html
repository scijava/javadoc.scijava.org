net.imagej.omero.legacy.text

Class OMEROText

  • java.lang.Object
    • net.imagej.omero.legacy.text.OMEROText
  • All Implemented Interfaces:
    Predicate<RealLocalizable>, OMERORealMask<TextData>, OMERORealMaskRealInterval<TextData>, EuclideanSpace, RealInterval, MaskPredicate<RealLocalizable>, RealMask, RealMaskRealInterval

    public class OMEROText
extends Object
implements OMERORealMaskRealInterval<TextData>
    Wraps an OMERO TextData as a RealMaskRealInterval.

    Technically text is not considered a ROI. This wrapper is only here to facilitate conversions between OMERO and ImageJ 1.x. As such it throw an UnsupportedOperationException for most RealMaskRealInterval methods. It should NOT be used for other purposes!
