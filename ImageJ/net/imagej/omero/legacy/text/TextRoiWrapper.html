    public class TextRoiWrapper
extends Object
implements IJRealRoiWrapper<ij.gui.TextRoi>
    An IJRealRoiWrapper which wraps a TextRoi as a RealMaskRealInterval.

    Technically text is not considered a ROI, as such UnsupportedOperationException is thrown for the majority of these methods. This wrapper is only here to provide a path for getting text from ImageJ 1.x and into OMERO. It should NOT be used for other purposes.
