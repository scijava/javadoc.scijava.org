Class MetadataUtil

  • java.lang.Object
    • net.imagej.ops.MetadataUtil

  • public class MetadataUtil
extends Object
    Utility class that allows to copy Metadata. It is the responsibility of the user to ensure that the source and target spaces have the correct dimensionality.
    Author:
    Christian Dietz (University of Konstanz), Michael Zinsmaier (University of Konstanz), Martin Horn (University of Konstanz)
