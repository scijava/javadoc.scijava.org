    • Method Detail

      • findMatch

        OpCandidate findMatch(OpEnvironment ops,
                      OpRef ref)
        Finds and initializes the best module matching the given op name and/or type + arguments. An OpCandidate containing the matching module will be returned.
        Parameters:
        ops - The pool from which candidate ops should be drawn.
        ref - The op reference describing the op to match.
        Returns:
        An OpCandidate containing the module which wraps the best Op, with populated inputs, ready to run.
        Throws:
        IllegalArgumentException - if there is no match, or if there is more than one match at the same priority.
