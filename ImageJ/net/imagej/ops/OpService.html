    DefaultOpService

    public interface OpService
extends PTService<Op>, ImageJService, OpEnvironment
    Interface for services that manage and execute Ops.

    Note that the op service implements OpEnvironment, to provide a "global" op execution environment. This is done for convenience of simple use cases, to make calling the built-in ops as easy as possible.
