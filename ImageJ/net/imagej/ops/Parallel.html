    public interface Parallel
    Base interface for Ops that run across multiple threads.

    A parallel op is one that executes across multiple threads. Typically, it parallelizes a Threadable op across those threads by relying on the Threadable.getIndependentInstance() method, but there is no requirement that parallel ops necessarily work in this way.
