net.imagej.ops.math

Class ConstantToArrayImageP

  • java.lang.Object
    • net.imagej.ops.math.ConstantToArrayImageP

  • public final class ConstantToArrayImageP
extends Object
    Wrapper class for multi-threaded version of binary math operations between constant values and ArrayImgs.

    Add, subtract, multiply, and divide are implemented for ArrayImgs of ByteType or DoubleType.

    Author:
    Leon Yang
