Interface Overlay

  • All Superinterfaces:
    AnnotatedSpace<CalibratedAxis>, CalibratedRealInterval<CalibratedAxis>, CalibratedSpace<CalibratedAxis>, org.scijava.Contextual, Data, net.imglib2.EuclideanSpace, org.scijava.Named, net.imglib2.RealInterval, TypedRealInterval<CalibratedAxis>, TypedSpace<CalibratedAxis>
    All Known Implementing Classes:
    AbstractOverlay, AbstractROIOverlay, AngleOverlay, BinaryMaskOverlay, CompositeOverlay, EllipseOverlay, GeneralPathOverlay, LineOverlay, PointOverlay, PolygonOverlay, RectangleOverlay, TextOverlay

    public interface Overlay
extends Data
    An overlay is a vector object, typically displayed with Datasets. Many (but not all) overlays have an associated ImgLib RegionOfInterest allowing iteration over pixel values deemed included in the overlay.
    Author:
    Lee Kamentsky, Curtis Rueden
