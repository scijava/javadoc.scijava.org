Class Binarize<T extends net.imglib2.type.numeric.RealType<T>>

  • java.lang.Object
    • org.scijava.AbstractContextual
      • org.scijava.command.ContextCommand
        • net.imagej.plugins.commands.binary.Binarize<T>
  • All Implemented Interfaces:
    Runnable, org.scijava.Cancelable, org.scijava.command.Command, org.scijava.Contextual, org.scijava.plugin.SciJavaPlugin

    public class Binarize<T extends net.imglib2.type.numeric.RealType<T>>
extends org.scijava.command.ContextCommand
    Creates or updates a mask Dataset of type BitType. Uses an input Dataset that will be thresholded. Thresholding options can be specified which are used to discriminate pixel values. The input Dataset can become the mask if desired.
    Author:
    Barry DeZonia
