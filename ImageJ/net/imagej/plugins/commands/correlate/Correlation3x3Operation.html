net.imagej.plugins.commands.correlate

Class Correlation3x3Operation

  • java.lang.Object
    • net.imagej.plugins.commands.correlate.Correlation3x3Operation

  • public class Correlation3x3Operation
extends Object
    Correlation3x3Operation is used for general 3x3 correlation. It takes a 3x3 kernel as input. Kernel is actually stored as a 1-D array such that {0,1,2,3,4,5,6,7,8} implies this shape: {{0,1,2},{3,4,5},{6,7,8}}. This class is used by the various Shadow implementations, SharpenDataValues, SmoothDataValues, etc.
