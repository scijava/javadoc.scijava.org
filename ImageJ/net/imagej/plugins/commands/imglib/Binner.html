    public class Binner<T extends net.imglib2.type.numeric.RealType<T>>
extends org.scijava.command.ContextCommand
    Reduces the size of an image by integral scale factors. The scale factors can be manipulated separately per dimension (i.e. scale X by 2 but Y by 3). The resultant pixel is some combination (min, avg, sum, etc.) of the original neighboring pixels. The combination method can be specified as needed.
