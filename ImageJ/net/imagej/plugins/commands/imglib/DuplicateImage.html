      • setDefaultBehavior

        public void setDefaultBehavior(boolean value)
        Specifies whether to use default behavior or special behavior. Default behavior copies the current composite XY plane. Special behavior is set when user defined axis definitions are to follow. If no axis definitions follow then all planes within the selected region will be copied.

      • isDefaultBehavior

        public boolean isDefaultBehavior()
        Returns true if current behavior is default. Otherwise current behavior is special. Default behavior copies the current composite XY plane. When special behavior is set then user defined axis definitions may have been specifed and if so those definitions are used during copying. Otherwise if no user defined axis definitions were specified then all planes within the selected region will be copied.

      • setAxisRange

        public String setAxisRange(AxisType axis,
                           String axisDefinition,
                           boolean originIsOne)
        Sets the range of values to copy from the input display for the given axis. The definition is a textual language that allows one or more planes to be defined by one or more comma separated values. Some examples:

        • "1" : plane 1
        • "3,5" : planes 3 and 5
        • "1-10" : planes 1 through 10
        • "1-10,20-30" : planes 1 through 10 and 20 through 30
        • "1-10-2,20-30-3" : planes 1 through 10 by 2 and planes 20 through 30 by 3
        • "1,3-5,12-60-6" : this shows each type of specification can be combined

        Returns:
        null if successful else a message describing input error
