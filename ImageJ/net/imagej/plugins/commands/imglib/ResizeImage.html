    public class ResizeImage<T extends net.imglib2.type.numeric.RealType<T>>
extends org.scijava.command.ContextCommand
    Resamples an existing image into a Dataset of specified dimensions. The dimensions can be manipulated separately. The resultant pixel is some combination of the original neighboring pixels using some user specified interpolation method.
