Class ResliceImage<T extends RealType<T>>

  • All Implemented Interfaces:
    Runnable, org.scijava.Cancelable, org.scijava.command.Command, org.scijava.Contextual, org.scijava.plugin.SciJavaPlugin

    public class ResliceImage<T extends RealType<T>>
extends org.scijava.command.ContextCommand
    Updates dimensions and data values of an image Dataset. The user supplies spacings for each dimension. The new data is the result of slicing the original image at specified spacings. When spacings are not equal to integral spacings the resulting data has different dimensions. The output data is computed using settable interpolation methods.

    Barry DeZonia
