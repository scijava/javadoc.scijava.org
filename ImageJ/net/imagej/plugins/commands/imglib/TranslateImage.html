    public class TranslateImage<T extends net.imglib2.type.numeric.RealType<T>>
extends org.scijava.command.ContextCommand
    Translates an existing image by a set of possibly nonintegral deltas. The deltas can be manipulated by dimension. The resultant pixel is some combination of the original neighboring pixels using some user specified interpolation method.

    Barry DeZonia
