    public class ReorderData
extends DynamicCommand
    Changes the storage order of the pixels of a Dataset. One can change the order in many ways (such as from [x,y,c,z,t] to [x,y,t,c,z]). The pixel data and axis labels are rearranged to match the new axis order.
