    public class TypeChanger<U extends RealType<U>,V extends RealType<V> & NativeType<V>>
extends DynamicCommand
    TypeChanger changes the type of the data in a Dataset. The DataType of the data is chosen from the types discovered at runtime by the DataTypeService. Channels can be combined in the process if desired. Combination is done via channel averaging. After conversion data values are preserved as much as possible but do get clamped to the new data type's valid range.
