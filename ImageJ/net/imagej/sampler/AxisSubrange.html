net.imagej.sampler

Class AxisSubrange

  • java.lang.Object
    • net.imagej.sampler.AxisSubrange

  • public class AxisSubrange
extends Object
    An AxisSubrange defines a set of position indices using various constructors. A set of position indices might look like this: 1, 3, 6, 7, 8, 25, 44. These indices are used to iterate over a subset of an image.

      • AxisSubrange

        public AxisSubrange(ImageDisplay display,
                    AxisType axis,
                    String definition,
                    boolean originOne)
        Create an AxisSubrange from a String definition. The definition is a textual language that allows one or more positions to be defined by one or more comma separated values. Some examples:

        • "1" : plane 1
        • "3,5" : planes 3 and 5
        • "1-10" : planes 1 through 10
        • "1-10,20-30" : planes 1 through 10 and 20 through 30
        • "1-10-2,20-30-3" : planes 1 through 10 by 2 and planes 20 through 30 by 3
        • "1,3-5,12-60-6" : an example combining all three formats
        Parameters:
        display - The ImageDisplay used to set legal bounds on dimension sizes
        axis - The AxisType within the ImageDisplay that defines bounds
        definition - The textual description of the subrange (detailed elsewhere)
        originOne - A boolean specifying whether the origin within the String language should be treated as 0 based or 1 based.
