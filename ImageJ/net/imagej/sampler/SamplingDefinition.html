Class SamplingDefinition

  • java.lang.Object
    • net.imagej.sampler.SamplingDefinition

  • public class SamplingDefinition
extends Object
    SamplingDefinitions define regions of space and are used by the SamplerService to pull data out of existing ImageDisplays.

    Barry DeZonia

      • getInputRanges

        public List<List<Long>> getInputRanges()
        Returns a multidimensional set of input axis values generated from the input data of this SamplingDefinition.

        For example, if the sampling definition has two axes defined as "1-4" and "1-3" calling this routine would return something like this: [[1,2,3,4] , [1,2,3]]

public double[] getOutputCalibration(AxisType[] outputAxes)

      • constrain

        Parameters:
