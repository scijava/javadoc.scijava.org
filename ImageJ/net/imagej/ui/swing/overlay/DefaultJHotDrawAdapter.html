    public class DefaultJHotDrawAdapter
extends AbstractJHotDrawAdapter<Overlay,ImageFigure>
    The default adapter handles any kind of overlay. It uses the fill color and alpha of the overlay to draw the mask and leaves the rest of the figure transparent.

    Lee Kamentsky
