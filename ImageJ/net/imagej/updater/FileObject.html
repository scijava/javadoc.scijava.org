net.imagej.updater

Class FileObject

  • java.lang.Object
    • net.imagej.updater.FileObject

  • public class FileObject
extends Object
    This class represents a file handled by the updater.

    The ImageJ updater knows about certain files (see Checksummer.directories for details). These files can be local-only, up-to-date, updateable, etc.

    Author:
    Johannes Schindelin
