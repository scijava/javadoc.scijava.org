Class FileUploader

    public class FileUploader
extends AbstractUploader
    This FileUploader is highly specialized to upload files and XML information over to ImageJ2 Update sites. There is a series of steps to follow. Any exception means entire upload process is considered invalid.
    1. Set db.xml.gz to read-only
    2. Verify db.xml.gz has not been modified, if not, upload process cancelled
    3. Upload db.xml.gz.lock (Lock file, prevent others from writing it ATM)
    4. If all goes well, force rename db.xml.gz.lock to db.xml.gz

    Johannes Schindelin
