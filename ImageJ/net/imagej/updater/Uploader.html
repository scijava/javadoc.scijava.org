net.imagej.updater

Interface Uploader

  • All Superinterfaces:
    ImageJPlugin, Progress, Progressable, org.scijava.plugin.SciJavaPlugin
    All Known Implementing Classes:
    AbstractUploader, FileUploader, SFTPFileUploader, SSHFileUploader, WebDAVUploader

    public interface Uploader
extends ImageJPlugin, Progressable
    Interface for ImageJ upload mechanisms.

    Upload mechanisms discoverable at runtime must implement this interface and be annotated with @Plugin with attribute Plugin.type() = Uploader.class. While it possible to create an upload mechanism merely by implementing this interface, it is encouraged to instead extend AbstractUploader, for convenience.

    Author:
    Johannes Schindelin, Curtis Rueden
