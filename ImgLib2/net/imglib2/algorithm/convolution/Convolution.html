net.imglib2.algorithm.convolution

Interface Convolution<T>

  • All Known Implementing Classes:
    AbstractMultiThreadedConvolution, LineConvolution, MultiDimensionConvolution

    public interface Convolution<T>
    This interface allows the client to perform a convolution. But also to query for the required input image size and preferred input image type. The ExectorService can be set, to allow multi or single threaded operation.

    Very importantly, multiple Convolutions can be easily concatenated.

    Author:
    Matthias Arzt
