    @Deprecated
public class FourierConvolution<T extends RealType<T>,S extends RealType<S>>
extends Object
implements MultiThreaded, OutputAlgorithm<Img<T>>, Benchmark
    Computes a convolution of an Img or RandomAccessibleInterval with an kernel. The computation is based on the Fourier convolution theorem and computation time is therefore independent of the size of kernel (except the kernel becomes bigger than the input, which makes limited sense). It is possible to exchange the kernel or the image if a series of images is convolved with the same kernel - or if an image has to be convolved with multiple kernels. The precision of the computation is ComplexFloatType.
    Author:
    Stephan Preibisch
