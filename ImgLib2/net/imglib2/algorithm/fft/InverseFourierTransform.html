Class InverseFourierTransform<T extends RealType<T>,S extends ComplexType<S>>

  • java.lang.Object
    • net.imglib2.algorithm.fft.InverseFourierTransform<T,S>
  • Type Parameters:
    T - - the output, RealType
    S - - the input, ComplexType
    All Implemented Interfaces:
    Algorithm, Benchmark, MultiThreaded, OutputAlgorithm<Img<T>>
    Deprecated. 
    use FFT instead

    @Deprecated
public class InverseFourierTransform<T extends RealType<T>,S extends ComplexType<S>>
extends Object
implements MultiThreaded, OutputAlgorithm<Img<T>>, Benchmark
    Computes the inverse Fourier Transform of a given RandomAccessibleInterval or Img. The inverse transform is always performed in place as otherwise more constraints for the input are necessary (has to be copied). If you do not want to compute it in place, make a copy before.
    Author:
    Stephan Preibisch
