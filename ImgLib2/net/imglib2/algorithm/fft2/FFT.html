net.imglib2.algorithm.fft2

Class FFT

  • java.lang.Object
    • net.imglib2.algorithm.fft2.FFT

  • public class FFT
extends Object
    Compute a FFT transform, either real-to-complex, complex-to-complex, or complex-to-real for an entire dataset. Unfortunately only supports a maximal size of INT in each dimension as the one-dimensional FFT is based on arrays.
    Author:
    Stephan Preibisch
