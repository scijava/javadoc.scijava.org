Class LevenbergMarquardtSolver

    public class LevenbergMarquardtSolver
extends Object
implements FunctionFitter
    A plain implementation of Levenberg-Marquardt least-square curve fitting algorithm. This solver makes use of only the function value and its gradient. That is: candidate functions need only to implement the FitFunction.val(double[], double[]) and FitFunction.grad(double[], double[], int) methods to operate with this solver.

    It was adapted and stripped from jplewis (www.idiom.com/~zilla) and released under the GPL. There are various small tweaks for robustness and speed.

    Author:
    Jean-Yves Tinevez 2011 - 2013
