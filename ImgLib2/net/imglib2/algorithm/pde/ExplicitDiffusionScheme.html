      • setDiffusionTensor

        public void setDiffusionTensor(RandomAccessibleInterval<FloatType> D)
        Set the diffusion tensor that will be used for the diffusion process.

        The diffusion tensor must be a FloatType RandomAccessibleInterval, with specific dimensions: If the target image has N dimensions, the tensor must have N+1 dimensions with:

        • the first N dimensions size equal to the input size;
        • the N+1 dimension having a size of 3.

        The tensor stores the local diffusion intensity and orientation in the shape of a real symmetric 2x2 matrix. Along the N+1 dimension, the tensor components are ordered as:

        1. Dxx
        2. Dxy
        3. Dyy
