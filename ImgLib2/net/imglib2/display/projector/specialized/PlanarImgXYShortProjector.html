        public PlanarImgXYShortProjector(PlanarImg<A,ShortArray> source,
                                 ArrayImg<UnsignedShortType,ShortArray> target,
                                 double normalizationFactor,
                                 double min)
        Normalizes a PlanarImg and writes the result into target. This can be used in conjunction with UnsignedByteAWTScreenImage for direct displaying. The normalization is based on a normalization factor and a minimum value with the following dependency:

        normalizationFactor = (typeMax - typeMin) / (newMax - newMin)
        min = newMin

        A value is normalized by: normalizedValue = (value - min) * normalizationFactor.
        Additionally the result gets clamped to the type range of target (that allows playing with saturation...).
        Parameters:
        source - Signed/Unsigned input data
        target - Unsigned output

        min -
