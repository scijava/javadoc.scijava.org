      • getPixelsZeroBasedIndex

        protected abstract Object getPixelsZeroBasedIndex(int index)
        This method is used internally by AbstractVirtualStack to implement getPixels(int).

        Returns an array that contains the pixels of the specified XY-plane. The type of the array must be byte[], short[], int[] or float[] and must match the type indicated by getBitDepth().

        Parameters:
        index - Zero based index of the plane. (Warning VirtualStack.getPixels(int) uses one base indices).
        See Also:
        ImageStack.getBitDepth()

      • setPixelsZeroBasedIndex

        protected abstract void setPixelsZeroBasedIndex(int index,
                                                Object pixels)
        This method is used internally by AbstractVirtualStack to implement setPixels(java.lang.Object, int).

        Set all pixels in the specified XY-plane.

        Parameters:
        index - Zero based index of the plane. (Warning VirtualStack.getPixels(int) uses one base indices).
        pixels - Array that contains the pixel data. Array length must equal getWidth() times getHeight(). Type of the array must be byte[], short[], int[] or float[] and match the type indicated by getBitDepth().

      • getSliceZeroBasedIndex

        protected RandomAccessibleInterval<?> getSliceZeroBasedIndex(int index)
        This method is used internally by AbstractVirtualStack to implement getVoxels(int, int, int, int, int, int, float[]) and setVoxels(int, int, int, int, int, int, float[]).

        Returns a two dimensional RandomAccessibleInterval that provides read and write access to the specified XY-plane. The pixel type must be match getBitDepth(). RealType for 8, 16 and 32 bit images, and ARGBType fot 32-bit images.

        Parameters:
        index - Zero based index of the plane. (Warning VirtualStack.getPixels(int) uses one base indices).
