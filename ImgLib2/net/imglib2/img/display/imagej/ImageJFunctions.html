net.imglib2.img.display.imagej

Class ImageJFunctions

  • java.lang.Object
    • net.imglib2.img.display.imagej.ImageJFunctions

  • public class ImageJFunctions
extends Object
    Provides static convenience methods to facilitate interacting with ImageJ 1.x.

    When interacting between ImageJ 1.x and ImgLib2, it is desirable to adapt data structures instead of copying data.

    For example, when an ImagePlus is made available by ImageJ 1.x, you can pass it to ImgLib2 as an Img via ImageJFunctions.wrap(imp).

    Likewise, when an ImgLib2 RandomAccessibleInterval needs to be passed to ImageJ 1.x, it can be wrapped into an ImagePlus via ImageJFunctions.wrap(img, title).

    Author:
    Tobis Pietzsch, Stephan Preibisch, Stephan Saalfeld
