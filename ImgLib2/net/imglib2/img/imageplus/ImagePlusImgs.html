net.imglib2.img.imageplus

Class ImagePlusImgs

  • java.lang.Object
    • net.imglib2.img.imageplus.ImagePlusImgs

  • public final class ImagePlusImgs
extends Object
    Convenience factory methods for creation of ImagePlusImg instances with the most common pixel Type variants. Keep in mind that this cannot be a complete collection since the number of existing pixel Types may be extended. For pixel Types T not present in this collection, use the generic ImagePlusImgFactory.create(long[], net.imglib2.type.NativeType), e.g. 
     img = new ImagePlusImgFactory< MyType >.create( new long[] { 100, 200 }, new MyType() );
    Author:
    Stephan Saalfeld
