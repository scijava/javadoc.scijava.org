Class AbstractAffineTransform

  • java.lang.Object
    • net.imglib2.realtransform.AbstractAffineTransform
  • All Implemented Interfaces:
    EuclideanSpace, AffineGet, AffineSet, InvertibleRealTransform, RealTransform
    Direct Known Subclasses:
    AffineTransform

    public abstract class AbstractAffineTransform
extends Object
implements AffineGet, AffineSet
    An abstract implementation of an affine transformation that returns default values referring to the identity transformation for all fields. This implementation is not thread safe. Create a InvertibleRealTransform.copy() for each consumer.
    Author:
    Stephan Saalfeld
