Class ScaleAndTranslation

  • java.lang.Object
    • net.imglib2.realtransform.ScaleAndTranslation
  • All Implemented Interfaces:
    Concatenable<ScaleAndTranslationGet>, PreConcatenable<ScaleAndTranslationGet>, EuclideanSpace, AffineGet, InvertibleRealTransform, RealTransform, ScaleAndTranslationGet

    public class ScaleAndTranslation
extends Object
implements ScaleAndTranslationGet, Concatenable<ScaleAndTranslationGet>, PreConcatenable<ScaleAndTranslationGet>
    An n transform that applies a scaling first and then shifts coordinates. This transform is faster than using an AffineTransform with entries on the diagonal only.
    Author:
    Philipp Hanslovsky, Stephan Saalfeld
