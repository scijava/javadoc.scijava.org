Class BacktrackingLineSearch

  • java.lang.Object

  • public class BacktrackingLineSearch

    A generic backtracking line search to iteratively estimate the inverse of a RealTransform. This should never be used when a closed-form inverse is available.

      • iterativeInverse

        public double iterativeInverse(double[] source,
                               double[] destination)
        Find the source point that, when pushed through the forward transform, results in the given destination point. Iteratively estimates the result with backtracking line search.

        Terminates either when a maximum iteration count has been reached, or the error falls below a specified threshold.

        Parameters:
        source - the source point that will be overwritten
        destination - the destination point
        Returns:
        error of inverse
