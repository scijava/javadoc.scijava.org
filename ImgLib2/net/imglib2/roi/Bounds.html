Class Bounds<I extends RealInterval,B extends Bounds<I,B>>

  • java.lang.Object
    • net.imglib2.roi.Bounds<I,B>
  • Type Parameters:
    I - interval type (Interval or RealInterval)
    B - recursive type of this Bounds
    Direct Known Subclasses:
    Bounds.IntBounds, Bounds.RealBounds

    public abstract class Bounds<I extends RealInterval,B extends Bounds<I,B>>
extends Object
    Operations on mask bounds. Bounds can be UNBOUNDED, or a (possibly empty) (Real)Interval. Bounds can be composites of other Bounds and change if the underlying Bounds are modified. A Bounds can never go from UNBOUNDED to being an interval or vice versa, though.

    Specialized for RealInterval and Interval in nested subclasses Bounds.IntBounds and Bounds.RealBounds.
