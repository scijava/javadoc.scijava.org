      • equals

        boolean equals(Object obj)
        Determines whether two masks describe the same region in the same way. Each kind of mask has its own semantics:
        • Two masks of the same kind describe the same region when they have the same attributes; for example, two polygon objects are equal if they have the same vertices in the same order at the same locations.
        • Two masks of different kinds are never considered to be describing the same region. For example, a box is not considered "equal" to a polygon even if the polygon's vertices are the four corners of the box, and the two masks have the same boundary type.
