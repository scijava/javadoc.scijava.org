      • Process

        public Process(Class<Algorithm> algorithmClass,
               Img<? extends RealType<?>> img,
               Object... parameters)
        throws Exception
        Initialize and execute the given Algorithm and prepare a cursor to deliver its pixel values one by one in successive calls to eval(). If the @param algorithmClass implements OutputAlgorithm, then this IFunction will iterate over the image obtained from getResult(). Otherwise, this IFunction iterates over the modified input image. The Algorithm is constructed by reflection and assumes that the constructor accepts one Image as first argument and then the rest of the argument in the same order as given. Otherwise, an Exception will be thrown.
        Throws:
        Exception
