    public class Gamma
extends AbstractFilterFn
    Return a function that, when evaluated, computes the gamma of the given image. If 'i' was the pixel value, and the image was 8-bit, then this function would do: double v = Math.exp(Math.log(i/255.0) * gamma) * 255.0); if (v < 0) v = 0; if (v >255) v = 255;
