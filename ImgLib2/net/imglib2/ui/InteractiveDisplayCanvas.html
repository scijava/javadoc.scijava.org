Interface InteractiveDisplayCanvas<A>

  • Type Parameters:
    A - transform type
    All Superinterfaces:
    TransformListener<A>
    All Known Implementing Classes:
    InteractiveDisplayCanvasComponent

    public interface InteractiveDisplayCanvas<A>
extends TransformListener<A>
    A component that uses one or several OverlayRenderers to render a canvas displayed on screen. Current implementations derive from JComponent.

    InteractiveDisplayCanvas also owns a TransformEventHandler, which is registered to listen to mouse and keyboard events if it implements MouseListener, etc.

    Moreover, InteractiveDisplayCanvas is a transform event multi-caster. It receives transformChanged events (usually from its TransformEventHandler) and propagates them to all registered listeners.

    Author:
    Tobias Pietzsch
