net.imglib2.ui

Interface RenderSource<T,A>

  • Type Parameters:
    T - pixel type
    A - transform type
    All Known Implementing Classes:
    FinalSource, InterpolatingSource

    public interface RenderSource<T,A>
    Image data to be rendered by imglib2-ui. Consists of a RealRandomAccessible of arbitrary pixel type T, a source transform, and an converter that converts type T to ARGBType for display. The source accessible is assumed to be extended to infinity. If the underlying data is a pixel image, it is also assumed to be interpolated already (see InterpolatingSource).

    Tobias Pietzsch
