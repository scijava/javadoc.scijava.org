Interface RenderTarget

  • All Known Implementing Classes:
    BufferedImageOverlayRenderer

    public interface RenderTarget
    Receiver for a BufferedImage (to be drawn onto a canvas later).

    A Renderer will render source data into a BufferedImage and provide this to a RenderTarget.

    See BufferedImageOverlayRenderer, which is a RenderTarget and also an OverlayRenderer that draws the BufferedImage.

    Author:
    Tobias Pietzsch
