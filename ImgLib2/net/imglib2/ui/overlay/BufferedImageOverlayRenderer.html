    • Field Detail

      • bufferedImage

        protected BufferedImage bufferedImage
        The BufferedImage that is actually drawn on the canvas. Depending on Defaults.discardAlpha this is either the BufferedImage obtained from screen image, or screen image's buffer re-wrapped using a RGB color model.

      • pendingImage

        protected BufferedImage pendingImage
        A BufferedImage that has been previously set for painting. Whenever a new image is set, this is stored here and marked pending. Whenever an image is painted and a new image is pending, the new image is painted to the screen. Before doing this, the image previously used for painting is swapped into pendingImage. This is used for double-buffering.
