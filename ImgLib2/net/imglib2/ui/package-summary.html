  • Interface Summary 
    Interface Description
    AffineTransformType<A>
    Provide create and set operations on (affine) transforms of type A.
    InteractiveDisplayCanvas<A>
    A component that uses one or several OverlayRenderers to render a canvas displayed on screen.
    InterruptibleProjector
    Similar to a Projector, this renders a target image (usually a 2D RandomAccessibleInterval).
    OverlayRenderer
    Draw something to a Graphics canvas and receive notifications about changes of the canvas size.
    PainterThread.Paintable  
    Renderer<A>
    Renders some source data applying a viewer transform (mapping global to screen coordinates).

    Factory for TransformEventHandler.
