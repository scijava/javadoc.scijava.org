Class InterpolatingSource<T extends NumericType<T>,A>

  • java.lang.Object
    • net.imglib2.ui.util.InterpolatingSource<T,A>
  • Type Parameters:
    T - pixel type
    A - transform type
    All Implemented Interfaces:
    RenderSource<T,A>

    public class InterpolatingSource<T extends NumericType<T>,A>
extends Object
implements RenderSource<T,A>
    A RenderSource, that provides an interpolated source RandomAccessible and is able to switch between nearest-neighbor and n-linear interpolation. The (discrete) source RandomAccessible, transform, and Converter provided in the constructor.

    Tobias Pietzsch
