    public class InteractiveRealViewer<A extends AffineSet & AffineGet & Concatenable<AffineGet>,C extends JComponent & InteractiveDisplayCanvas<A>>
extends Object
implements TransformListener<A>, PainterThread.Paintable
    Simple interactive viewer window. It creates a JFrame with the given canvas, and sets up transformation handling and rendering.

    It implements PainterThread.Paintable to handle repainting through a PainterThread. It implements TransformListener to be notified about viewer transformation changes made by the user.

    Author:
    Tobias Pietzsch

      • InteractiveRealViewer

        public InteractiveRealViewer(AffineTransformType<A> transformType,
                             C interactiveDisplayCanvas,
                             RendererFactory<A> rendererFactory)
        Create an interactive viewer window displaying the specified interactiveDisplayCanvas, and create a Renderer which draws to that canvas.

        A Renderer is created that paints to a BufferedImageOverlayRenderer render target which is displayed on the canvas as an OverlayRenderer. A PainterThread is created which queues repainting requests from the renderer and interactive canvas, and triggers repainting of the viewer.

        Parameters:
        transformType -
        interactiveDisplayCanvas - the canvas JComponent which will show the rendered images.
        rendererFactory - is used to create the Renderer.
