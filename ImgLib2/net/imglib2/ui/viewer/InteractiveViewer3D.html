        Create an interactive viewer for a 3D RandomAccessible.
        Parameters:
        width - window width.
        height - window height.
        source - The source image to display. It is assumed that the source is extended to infinity.
        sourceInterval - The size of the source in source local coordinates. This is used for displaying a navigation wire-frame cube.
        sourceTransform - Transformation from source to global coordinates. This is useful for pre-scaling when showing anisotropic data, for example.
        converter - Converter from the source type to argb for rendering the source.
