Class PickConeRay

  • java.lang.Object
    • org.scijava.java3d.PickShape

        • org.scijava.java3d.PickConeRay

  • public final class PickConeRay
extends PickCone
    PickConeRay is an infinite cone ray pick shape. It can be used as an argument to the picking methods in BranchGroup and Locale.
    Since:
    Java 3D 1.2
    See Also:
    BranchGroup.pickAll(org.scijava.java3d.PickShape), Locale.pickAll(org.scijava.java3d.PickShape)
