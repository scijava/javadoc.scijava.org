org.scijava.java3d

Class ShaderError

  • java.lang.Object
    • org.scijava.java3d.ShaderError

  • public class ShaderError
extends Object
    ShaderError is a container object that holds the details of a runtime error that occurs while compiling or executing a programmable shader.
    Since:
    Java 3D 1.4
