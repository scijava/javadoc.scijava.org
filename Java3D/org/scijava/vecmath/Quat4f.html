Class Quat4f

  • java.lang.Object
    • org.scijava.vecmath.Tuple4f

    public class Quat4f
extends Tuple4f
implements Serializable
    A 4 element unit quaternion represented by single precision floating point x,y,z,w coordinates. The quaternion is always normalized.
