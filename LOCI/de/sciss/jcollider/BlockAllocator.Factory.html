      • create

        BlockAllocator create(int size,
                      int pos)
        Creates a new block allocator instance of the given size, with Bus indices starting at a given index.
        Parameters:
        size - the number of channels the allocator manages
        pos - the first bus index to use. this is usefull for example to return private audio busses which begin right after the hardware wired busses.
        Returns:
        a new empty allocator of the given size, starting at the given index
