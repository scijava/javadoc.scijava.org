Class ContiguousBlockAllocator

  • java.lang.Object
    • de.sciss.jcollider.ContiguousBlockAllocator
  • All Implemented Interfaces:
    BlockAllocator

    public class ContiguousBlockAllocator
extends Object
implements BlockAllocator
    Quite a 1:1 translation from the SClang class written by James Harkins, this class is a robust allocator for busses in a dynamic environment. It does not exhibit the fragmentation problem of the PowerOfTwoAllocator.

    0.32, 25-Feb-08
