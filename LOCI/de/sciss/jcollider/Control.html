Class Control

    public class Control
extends UGen
    Class for the Control UGen. This class is recognized by SynthDef in the building process. Controls should be created en bloc per rate (scalar or control), therefore the constructor takes an array of names and corresponding default values. For the sake of clarity, scalar control names should begin with a small 'i' letter, though this is not obligatory.

    Individual controls are accessed by calling the getChannel method.

    Note that when reading a synthdef from harddisc, the SynthDef class (as of version 0.25) will not create Control instances but merely plain UGen objects.
