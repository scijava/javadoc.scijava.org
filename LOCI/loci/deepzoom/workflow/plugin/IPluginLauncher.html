loci.deepzoom.workflow.plugin

Interface IPluginLauncher

  • All Known Implementing Classes:
    PluginLauncher

    public interface IPluginLauncher
    Interface to the plugin launcher. Each ILinkedPlugin is associated with a plugin class and has a IPluginLauncher that is uses internally to chain its plugin. The plugin launcher communicates with the plugin scheduler.
    Author:
    Aivar Grislis
