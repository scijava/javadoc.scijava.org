loci.multiinstanceplugin

Interface ILinkedPlugin

  • All Known Implementing Classes:
    LinkedPlugin

    public interface ILinkedPlugin
    Provides an interface to chain plugins together. The ILinkedPlugin instance is associated with a plugin class. It has an IPluginLauncher that it uses to chain plugins.
    Author:
    Aivar Grislis
