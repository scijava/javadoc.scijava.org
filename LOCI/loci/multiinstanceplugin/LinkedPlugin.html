Class LinkedPlugin

  • java.lang.Object
    • loci.multiinstanceplugin.LinkedPlugin
  • All Implemented Interfaces:
    ILinkedPlugin

    public class LinkedPlugin
extends Object
implements ILinkedPlugin
    The LinkedPlugin represents a given plugin class in a given stage of a pipeline. It listens for the set of input images that is required and launches instances of the plugin once that set is acquired.
    Author:
    Aivar Grislis
