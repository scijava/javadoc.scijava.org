Class WindowManager

  • java.lang.Object
    • loci.visbio.LogicManager
      • loci.visbio.WindowManager
  • All Implemented Interfaces:
    WindowListener, EventListener, Saveable

    public class WindowManager
extends LogicManager
implements WindowListener
    WindowManager is the manager encapsulating VisBio's window logic, including docking, resizing, minimization and cursor control.
