Class OverlayNodedObject

  • java.lang.Object
    • loci.visbio.overlays.OverlayObject

    OverlayFreeform, OverlayPolyline

    public abstract class OverlayNodedObject
extends OverlayObject
    OverlayNodedObject is an overlay object built of nodes likely connected by lines.
