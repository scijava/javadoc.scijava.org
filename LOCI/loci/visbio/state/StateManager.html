      • mergeStates

        public static void mergeStates(Vector oldList,
                               Vector newList)
        Merges states between two lists of dynamic objects. The procedure is as follows:
        1. Compare read object list with existing object list
        2. Find all matches between the two lists
        3. Reuse as many leftover existing objects as possible, initializing them to match compatible read objects
        4. If there are still leftover existing objects, discard them
        5. If there are still leftover read objects, initialize them
        See the Dynamic documentation for more information.
