loci.visbio.util

Class Docker

  • java.lang.Object
    • loci.visbio.util.Docker
  • All Implemented Interfaces:
    ComponentListener, EventListener

    public class Docker
extends Object
implements ComponentListener
    This class is used to specify a group of windows able to be docked to one another. When a window is moved close enough to another window, it snaps next to it, becoming docked. When the nearby window is subsequently moved, it drags all docked windows along with it. Docker also enables windows to snap to the edges of the screen, if desired.
