Class InstanceServer

  • java.lang.Object
    • loci.visbio.util.InstanceServer
  • All Implemented Interfaces:
    Runnable

    public class InstanceServer
extends Object
implements Runnable
    The InstanceServer provides a mechanism for managing the instances of an application running concurrently. The application should attempt to send any desired parameters to an already active instance server using the static sendArguments method. If successful (no exception thrown), the application should then shut down, having successfully passed its arguments to the active application instance. If unsuccessful (caught an exception), the application should create an InstanceServer, thus becoming the active application instance.
