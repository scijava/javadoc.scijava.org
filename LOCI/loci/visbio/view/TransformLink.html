Class TransformLink

  • java.lang.Object
    • loci.visbio.view.TransformLink
  • All Implemented Interfaces:

    public class TransformLink
extends Object
implements DisplayListener, Dynamic, Runnable, Saveable, TransformListener
    Represents a link between a data transform and a display that produces a single image.
