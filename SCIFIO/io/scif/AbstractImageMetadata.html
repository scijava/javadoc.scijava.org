      • setPlanarAxisCount

        public void setPlanarAxisCount(int count)
        Description copied from interface: ImageMetadata
        Sets the number of planar axes in this image. This value represents the number of dimensional axes constituting each Plane (as returned by the Reader.openPlane(int, long) methods). This value is necessary to determine the total plane count for an N-dimensional image, as well as how many pixels there are per plane.

        For example, suppose we have a 4-dimensional image with axes (X, Y, Z, T) and extents (768, 512, 7, 13). If there are two planar axes, then each plane is 768 x 512 and there are 7 x 13 = 91 total planes. But if we have three planar axes, then each plane is 768 x 512 x 7 and there are 13 total planes.

        Specified by:
        setPlanarAxisCount in interface ImageMetadata
        See Also:
        Reader.openPlane(int, long)

        public void setInterleavedAxisCount(int count)
        Description copied from interface: ImageMetadata
        Sets the number of interleaved axes in this image. This must be a value between [0, planarAxisCount). Interleaved axes are planar axes that do not constitute the "canonical" axes - e.g., in a CXY image with an interleaved axis count of 1, the C axis is interleaved, and each plane is an XY plane with C different representations.
