      • typedParse

        protected abstract void typedParse(org.scijava.io.handle.DataHandle<org.scijava.io.location.Location> handle,
                                   M meta,
                                   SCIFIOConfig config)
                            throws IOException,
                                   FormatException
        A helper method, called by parse(DataHandle, TypedMetadata). Allows for boilerplate code to come after parsing, specifically calls to Metadata.populateImageMetadata().

        This method should be implemented to populate any format-specific Metadata.

        NB: if a Format requires type-specific parsing to occur before the Abstract layer, Override #parse(String, TypedMetadata) instead.
