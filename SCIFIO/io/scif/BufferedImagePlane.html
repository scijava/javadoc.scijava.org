      • getBytes

        public byte[] getBytes()
        Standardizes this plane's BufferedImage to a byte[].

        NB: If this plane contains multiple channels, the planes for each channel will be condensed to a single array. The first entry for each channel's data will have an offset of: channelIndex * planeSize.

        Returns:
        The byte[] extracted from this Plane's BufferedImage
