io.scif

Class FilePattern

  • java.lang.Object
    • io.scif.FilePattern

  • public class FilePattern
extends Object
    FilePattern is a collection of methods for handling file patterns, a way of succinctly representing a collection of files meant to be part of the same data series. Examples:
    • C:\data\BillM\sdub<1-12>.pic
    • C:\data\Kevin\80<01-59>0<2-3>.pic
    • /data/Josiah/cell-Z<0-39>.C<0-1>.tiff
    Author:
    Curtis Rueden
