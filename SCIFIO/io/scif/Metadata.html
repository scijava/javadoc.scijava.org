    public interface Metadata
extends Serializable, HasFormat, HasSource, HasMetaTable
    Interface for all SCIFIO Metadata objects. Based on the format, a Metadata object can be a single N-dimensional collection of bytes (an image) or a list of multiple images.

    NB: When defining a new Metadata implementation, you will likely want to create a corresponding Format implementation. Also, at a minimum, a Translator should be implemented that can convert between Metadata and the new Metadata.

    See Also:
    Translator, AbstractFormat
