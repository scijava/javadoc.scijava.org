    public interface Reader
extends HasFormat, HasSource, Groupable
    Interface for all SCIFIO Readers.

    Reader components generate Plane representations of images via the openPlane(int, long) methods. These planes can then be used by calling software (e.g. for display) or passed to another method for writing to an output source (e.g. via the Writer.savePlane(int, long, io.scif.Plane) methods).

    Before a Reader can be used, it must be initialized via setSource(java.lang.String) and setMetadata(io.scif.Metadata) calls.

    Author:
    Mark Hiner
    See Also:
    Plane, Metadata, Writer.savePlane(int, long, io.scif.Plane)
