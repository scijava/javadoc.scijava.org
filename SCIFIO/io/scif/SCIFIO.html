    public class SCIFIO
extends org.scijava.AbstractGateway
    Convenience Context wrapper.

    Provides easy accessor methods for the SCIFIO-specific Service implementations.

    This class is intended to be a disposable wrapper that can easily be created and forgotten as needed.

    Author:
    Mark Hiner
