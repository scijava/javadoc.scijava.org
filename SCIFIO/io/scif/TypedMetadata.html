    public interface TypedMetadata
extends Metadata
    Sub-interface for Metadata to facilitate generic parameterization of SCIFIO components.

    Although Metadata itself does not require any generic parameters, this sub-interface allows type narrowing of method parameters that accept Metadata objects, e.g. Writer.setMetadata(Metadata) has a different type erasure than TypedWriter.setMetadata(TypedMetadata) so the two signatures can co-exist.
