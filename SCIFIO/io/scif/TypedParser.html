    public interface TypedParser<M extends TypedMetadata>
extends Parser
    Interface for all Parser implementations that use generic parameters.

    Generics allow each concrete Parser implementation to type narrow the return the type of Metadata from its parse(java.lang.String) methods, as well as the argument Metadata types for the same methods.
