    public interface TypedReader<M extends TypedMetadata,P extends DataPlane<?>>
extends Reader
    Interface for all Reader implementations that use generic parameters.

    Generics are used in Reader concrete implementations to type narrow return types, and to provide parallel methods that can type narrow arguments.

    Author:
    Mark Hiner
