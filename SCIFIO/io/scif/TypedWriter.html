    public interface TypedWriter<M extends TypedMetadata>
extends Writer
    Interface for all Writer implementations that use generic parameters.

    Generics all each concrete Writer implementation to guarantee, by type narrowing, any methods that return SCIFIO components. Also, parallel methods that take SCIFIO component arguments are defined to be type narrowed.

    Author:
    Mark Hiner
