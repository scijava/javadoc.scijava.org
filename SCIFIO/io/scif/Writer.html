    public interface Writer
extends HasFormat, HasSource
    Interface for all SCIFIO writers.

    Writer components are used to save Plane objects (output from the Reader.openPlane(int, long) methods) to a destination image location, via the savePlane(int, long, io.scif.Plane) methods.

    Author:
    Mark Hiner
    See Also:
    Plane, Reader.openPlane(int, long)

      • setMetadata

        void setMetadata(Metadata meta)
          throws FormatException
        Provides this Writer with a Metadata object to use when interpreting Planes during calls to savePlane(int, long, io.scif.Plane).

        NB: This method has accepts a general Metadata so that this signature can appear in the base interface for all Writers, but behavior if provided with a Metadata instance not associated with this Writer is undefined and should throw an exception.

        Parameters:
        meta - The Metadata to associate with this Writer.
        Throws:
        IllegalArgumentException - If the provided Metadata type is not compatible with this Writer.
        FormatException
