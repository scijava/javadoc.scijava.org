io.scif.SCIFIOService

  • Packages that use SCIFIOService 
    Package Description
    io.scif
    Provides the interfaces and default implementations for the components of SCIFIO, and the context for instantiating these components.
    io.scif.codec
    Codecs for encoding and decoding streams of bytes.
    io.scif.formats.qt  
    io.scif.formats.tiff
    This package contains utility classes for working with a variety of Tiff-based formats.
    io.scif.gui
    Provides classes for working with BufferedImage in SCIFIO.
    io.scif.img
    Provides SCIFIO-based classes for working with ImgPlus from the imglib2 library.
    io.scif.img.converters
    Provides a PlaneConverterService for populating ImgPlus objects using Reader output.
    io.scif.ome.services  
    io.scif.refs
    Provides reference management utilities.
    io.scif.services
    Provides a set of Services (singletons within a given context) for working with SCIFIO objects.
    io.scif.xml
    Provides tools and utility classes for working with XML in SCIFIO.
