    public abstract class AbstractReaderCommand
extends AbstractSCIFIOToolCommand
    Abstract superclass for SCIFIOToolCommand implementations that read images. Contains common parameter flags, e.g. for enabling certain filters, cropping, etc... methods for generating readers using these parameters, and a read helper method that performs all the necessary index setup and looping over desired planes.
