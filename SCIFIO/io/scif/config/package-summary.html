Package io.scif.config Description

This package provides configuration options for all SCIFIO components.

Configuration is achieved through use of the SCIFIOConfig class. Methods in this class are prefixed based on what component they are modifying. The SCIFIOConfig then operates as a single container of state, passed between each consumer, which queries the values that apply to them.

Configuration is scoped within method execution, thus SCIFIOConfig objects should not be cached, but instead always passed through method signatures.

Adding new configuration options requires three steps:

  • Add the appropriate field(s) and setter/accessor methods to SCIFIOConfig.
  • Modify the class being configured to accept SCIFIOConfig objects and query the desired information.
  • Update any downstream code of the newly configurable class to pass along SCIFIOConfig objects.
