    public abstract class AbstractMetadataWrapper
extends AbstractMetadata
implements MetadataWrapper
    Abstract superclass for concrete implementations of MetadataWrapper.

    To create a MetadataWrapper that is paired with a specific Filter, simply extend this class, override any methods as desired, and annotate the class using Plugin as appropriate. The wrapper will automatically be discovered and applied when a new instance of its Filter is instantiated, assuming the Filter is a subclass of AbstractReaderFilter
