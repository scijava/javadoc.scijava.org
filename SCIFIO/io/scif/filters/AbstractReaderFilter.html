    public abstract class AbstractReaderFilter
extends AbstractFilter<Reader>
implements Reader
    Abstract superclass for all Filter that delegate to Reader instances.

    NB: All concrete implementations of this interface should be annotated as Plugin for automatic discovery.

    NB: This class attempts to locate a type-matching MetadataWrapper to protectively wrap the wrapped Reader's Metadata. If none is found, a reference to the Reader's Metadata itself is used.

    Author:
    Mark Hiner
