  • All Implemented Interfaces:
    Filter, Groupable, HasFormat, HasSource, Reader, SCIFIOPlugin, Comparable<Prioritized>, Contextual, Identifiable, Locatable, Logged, HasPluginInfo, RichPlugin, SciJavaPlugin, Prioritized, Versioned

    public class ChannelFiller
extends AbstractReaderFilter
    For indexed color data representing true color, factors out the indices, replacing them with the color table values directly. For all other data (either non-indexed, or indexed with "false color" tables), does nothing. NB: lut length is not guaranteed to be accurate until a plane has been read
