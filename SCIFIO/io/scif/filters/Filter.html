    public interface Filter
extends SCIFIOPlugin
    Interface for modifying object behavior. Equivalent to wrapping/decorating.

    NB: Prioritized.getPriority() is intended to be used to determine a natural order for Comparable.compareTo(Object).

    Concrete Filter implementations should also extend the class they wrap, allowing the Filter to be used in the same way the wrapped class would be.

    Author:
    Mark Hiner
