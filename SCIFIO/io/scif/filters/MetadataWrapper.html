Interface MetadataWrapper

  • All Superinterfaces:
    Comparable<Prioritized>, Contextual, HasFormat, HasMetaTable, HasPluginInfo, HasSource, Identifiable, Locatable, Logged, Metadata, Prioritized, RichPlugin, SCIFIOPlugin, SciJavaPlugin, Serializable, Versioned
    All Known Implementing Classes:
    AbstractMetadataWrapper, ChannelFillerMetadata, DimensionSwapperMetadata, FileStitcherMetadata, PlaneSeparatorMetadata

    public interface MetadataWrapper
extends Metadata
    Wrapper for Metadata. Used to create defensive copies of metadata for manipulation by ReaderFilters, while allowing for API modification if needed.

    If a Reader-based Filter requires special functionality from its Metadata, a companion MetadataWrapper can be implemented. Concrete implementations of this interface should always be annotated with Plugin so they can be dynamically found when constructing new Filters.

    NB: This interface duplicates the Metadata setter signatures, with the addition of a passUp flag. If this flag is true, the wrapped metadata will also have the corresponding value set. If not, only the wrapper will be modified.
