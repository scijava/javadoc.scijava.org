      • getOriginalIndex

        public long getOriginalIndex(int imageIndex,
                             long planeIndex)
        Returns the image number in the original dataset that corresponds to the given image number. For instance, if the original dataset was a single RGB image and the given plane index is 2, the return value will be 0.
        Parameters:
        planeIndex - is a plane number greater than or equal to 0 and less than getPlaneCount()
        Returns:
        the corresponding plane number in the original (unseparated) data.
