Package io.scif.formats.tiff

This package contains utility classes for working with a variety of Tiff-based formats.

See: Description

  • Interface Summary 
    Interface Description
    TiffService
    Interface for services that work with TIFF files.
  • Class Summary 
    Class Description
    DefaultTiffService
    Default service for working with TIFF files.
    IFD
    Data structure for working with TIFF Image File Directories (IFDs).
    IFDList
    Data structure for working with a list of TIFF IFDs.
    OnDemandLongArray  
    TiffConstants
    Generally useful TIFF-related constants.
    TiffIFDEntry
    This class represents a single raw TIFF IFD entry.
    TiffParser
    Parses TIFF data from an input source.
    TiffRational
    A rational number (numerator over denominator).
    TiffSaver
    Writes TIFF data to an output location.
  • Enum Summary 
    Enum Description
    FillOrder
    Utility class for working with TIFF fill orders.
    IFDType
    An enumeration of IFD types.
    PhotoInterp
    Utility class for working with TIFF photometric interpretations.
    ResolutionUnit
    Utility class for working with TIFF resolution units.
    TiffCompression
    Utility class for performing compression operations with a TIFF file.
