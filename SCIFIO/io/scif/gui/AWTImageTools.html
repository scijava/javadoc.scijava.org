io.scif.gui

Class AWTImageTools

  • java.lang.Object
    • io.scif.gui.AWTImageTools

  • public final class AWTImageTools
extends Object
    A utility class with convenience methods for manipulating images in BufferedImage form. To work with images in primitive array form, use the ImageTools class. Much code was stolen and adapted from DrLaszloJamf's posts on the Java forums.
    Author:
    Curtis Rueden
