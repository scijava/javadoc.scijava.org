Package io.scif.gui

Provides classes for working with BufferedImage in SCIFIO.

See: Description

  • Interface Summary 
    Interface Description
    GUIService
    A Service for working with graphical user interfaces.
  • Class Summary 
    Class Description
    AWTImageTools
    A utility class with convenience methods for manipulating images in BufferedImage form.
    BufferedImageReader<M extends TypedMetadata>
    BufferedImageReader is the superclass for file format readers that use java.awt.image.BufferedImage as the native data type.
    BufferedImageTools
    Collection of static utility methods for working with BufferedImage.
    ComboFileFilter
    A file filter that recognizes files from a union of other filters.
    DefaultGUIService
    Basic GUIService implementation.
    FormatFileFilter
    A file filter for identifying the files supported by an associated Formats.
    ImageViewer
    A basic renderer for image data.
    Index16ColorModel
    ColorModel that handles 16 bits per channel lookup tables.
    PreviewPane
    PreviewPane is a panel for use as a JFileChooser accessory, displaying a thumbnail for the selected image, loaded in a separate thread.
    SignedByteBuffer
    DataBuffer that stores signed bytes.
    SignedColorModel
    ColorModel that handles 8, 16 and 32 bits per channel signed data.
    SignedShortBuffer
    DataBuffer that stores signed shorts.
    TwoChannelColorSpace
    ColorSpace for 2-channel images.
    UnsignedIntBuffer
    DataBuffer that stores unsigned ints.
    UnsignedIntColorModel
    ColorModel that handles unsigned 32 bit data.
    XYCTuple
    Helper class for converting N-D Metadata to x,y,c values for use in AWT classes.
