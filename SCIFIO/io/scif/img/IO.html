io.scif.img

Class IO

  • java.lang.Object
    • io.scif.img.IO

  • public final class IO
extends Object
    A static utility class for easy access to ImgSaver and ImgOpener methods. Also includes type-convenience methods for quickly opening various image types.

    NB: No exceptions are thrown by any methods in this class. Instead they are all caught, logged, and null is returned.

    NB: Each of these methods occurs in its own Context

    Author:
    Mark Hiner
