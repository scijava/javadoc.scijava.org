                                                                                 throws ImgIOException
        Reads in an ImgPlus from the given initialized Reader, using the given ImgFactory to construct the Img. The Type T to read is defined by the third parameter.

        NB: Any Reader provided must be wrapped by a PlaneSeparator filter.

        Parameters:
        reader - - An initialized Reader to use for reading image data.
        imgFactory - - The ImgFactory to use for creating the resultant ImgPlus.
        config - - SCIFIOConfig to use when opening this dataset
        Returns:
        - the ImgPlus or null
        Throws:
        ImgIOException - if there is a problem reading the image data.
