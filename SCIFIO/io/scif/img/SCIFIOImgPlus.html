    public class SCIFIOImgPlus<T>
extends ImgPlus<T>
implements Disposable
    SCIFIO extension of ImgPlus that adds getColorTable(int, int) API for specifying both the plane and image index.

    Also provides direct access to the Metadata object associated with the underlying dataset.
