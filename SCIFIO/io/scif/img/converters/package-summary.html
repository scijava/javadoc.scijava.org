Package io.scif.img.converters

Provides a PlaneConverterService for populating ImgPlus objects using Reader output.

See: Description

  • Interface Summary 
    Interface Description
    PlaneConverter
    Interface for using planes read by SCIFIO Readers to populate ImgPlus instances.
    PlaneConverterService
    Interface for requesting PlaneConverters to convert planes read by SCIFIO Readers to ImgPlus instances.
  • Class Summary 
    Class Description
    AbstractPlaneConverter
    Abstract superclass for PlaneConverter implementations.
    ArrayDataAccessConverter
    PlaneConverter implementation specialized for populating ArrayImg instances.
    DefaultPlaneConverterService
    Default PlaneConverterService implementation.
    PlanarAccessConverter
    PlaneConverter implementation specialized for populating PlanarAccess instances.
    RandomAccessConverter
    Generalized PlaneConverter implementation.
