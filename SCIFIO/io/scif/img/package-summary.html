Package io.scif.img Description

Provides SCIFIO-based classes for working with ImgPlus from the imglib2 library.

ImgOpener is used for creating ImgPluses, using CellImg or PlanarImg implementations.

ImgSaver is used for writing ImgPluses to disk.
