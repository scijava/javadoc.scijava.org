Package io.scif.io Description

Provides the classes for reading and writing information between image sources.

Changes since Bio-Formats

  • Added IStreamAccess interface. This interface is used as a middle ground for IRandomAccess implementations that will be stream based, and extracts many methods that were injected in an abstract layer in Bio-Formats.
  • The static functionality in Location has been extracted to LocationService

Future plans

  • {code IRandomAccess} naming. The IRandomAccess interface should define a getName and setName method, which will be implemented in the various handles.

    This value should be chained up to the RandomAccessInputStream toString method.

  • Remove id maps from Location. Should be able to construct with just the RandomAccessInputStream.
