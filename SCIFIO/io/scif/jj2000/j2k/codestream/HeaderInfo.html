Class HeaderInfo

  • java.lang.Object
    • io.scif.jj2000.j2k.codestream.HeaderInfo
  • All Implemented Interfaces:
    Markers, ProgressionType, FilterTypes, Cloneable

    public class HeaderInfo
extends Object
implements Markers, ProgressionType, FilterTypes, Cloneable
    Classe that holds information found in the marker segments of the main and tile-part headers. There is one inner-class per marker segment type found in these headers.
