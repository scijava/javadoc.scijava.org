io.scif.jj2000.j2k.wavelet

Class WTFilterSpec

  • java.lang.Object
    • io.scif.jj2000.j2k.wavelet.WTFilterSpec

  • public abstract class WTFilterSpec
extends Object
    This is the generic class from which the ones that hold the analysis or synthesis filters to be used in each part of the image derive. See AnWTFilterSpec and SynWTFilterSpec.

    The filters to use are defined by a hierarchy. The hierarchy is:

    - Tile and component specific filters
    - Tile specific default filters
    - Component main default filters
    - Main default filters

    At the moment tiles are not supported by this class.

    See Also:
    AnWTFilterSpec, SynWTFilterSpec
