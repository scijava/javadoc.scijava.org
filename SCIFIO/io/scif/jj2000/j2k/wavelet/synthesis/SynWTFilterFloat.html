Class SynWTFilterFloat

    Direct Known Subclasses:
    SynWTFilterFloatLift9x7

    public abstract class SynWTFilterFloat
extends SynWTFilter
    This extends the synthesis wavelet filter general definitions of SynWTFilter by adding methods that work for float data specifically. Implementations that work on float data should inherit from this class.

    See the SynWTFilter class for details such as normalization, how to split odd-length signals, etc.

    The advantage of using the specialized method is that no casts are performed.
