      • getMetadataStoreForDisplay

        public loci.formats.meta.MetadataStore getMetadataStoreForDisplay()
        Returns a MetadataStore that is populated in such a way as to produce valid OME-XML. The returned MetadataStore cannot be used by an IFormatWriter, as it will not contain the required BinData.BigEndian attributes.

      • getMetadataStoreForConversion

        public loci.formats.meta.MetadataStore getMetadataStoreForConversion()
        Returns a MetadataStore that is populated in such a way as to be usable by an IFormatWriter. Any OME-XML generated from this MetadataStore is very unlikely to be valid, as more than likely both BinData and TiffData element will be present.
