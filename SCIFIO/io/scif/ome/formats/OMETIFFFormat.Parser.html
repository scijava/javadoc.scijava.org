        Description copied from class: AbstractParser
        A helper method, called by AbstractParser.parse(RandomAccessInputStream, TypedMetadata). Allows for boilerplate code to come after parsing, specifically calls to Metadata.populateImageMetadata().

        This method should be implemented to populate any format-specific Metadata.

        NB: if a Format requires type-specific parsing to occur before the Abstract layer, Override #parse(String, TypedMetadata) instead.
