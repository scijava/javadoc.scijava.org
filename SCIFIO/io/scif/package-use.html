Uses of Package
io.scif

  • Packages that use io.scif 
    Package Description
    io.scif
    Provides the interfaces and default implementations for the components of SCIFIO, and the context for instantiating these components.
    io.scif.bf
    Contains Bio-Formats format, wrapping Bio-Formats proprietary readers as a SCIFIO Format.
    io.scif.bf.ome
    Contains translators from BioFormatsFormat to OME-XML.
    io.scif.cli  
    io.scif.cli.convert  
    io.scif.cli.help  
    io.scif.cli.info  
    io.scif.cli.show  
    io.scif.codec
    Codecs for encoding and decoding streams of bytes.
    io.scif.config
    This package provides configuration options for all SCIFIO components.
    io.scif.filters
    Provides wrapping classes for modifying the performance of one or more of a {link io.scif.Format}'s components.
    io.scif.formats
    Provides Format implementations for all file formats that are supported by default in SCIFIO.
    io.scif.formats.qt  
    io.scif.formats.tiff
    This package contains utility classes for working with a variety of Tiff-based formats.
    io.scif.gui
    Provides classes for working with BufferedImage in SCIFIO.
    io.scif.img
    Provides SCIFIO-based classes for working with ImgPlus from the imglib2 library.
    io.scif.img.cell
    Provides a Reader backed CachedCellImg implementation and supporting classes.
    io.scif.img.cell.loaders
    Provides implementations of SCIFIOArrayLoader for each basic ArrayDataAccess type.
    io.scif.img.converters
    Provides a PlaneConverterService for populating ImgPlus objects using Reader output.
    io.scif.lifesci  
    io.scif.ome  
    io.scif.ome.commands  
    io.scif.ome.services  
    io.scif.ome.translators  
    io.scif.refs
    Provides reference management utilities.
    io.scif.services
    Provides a set of Services (singletons within a given context) for working with SCIFIO objects.
    io.scif.util
    Provides utility classes for common operations on or by SCIFIO components.
    io.scif.xml
    Provides tools and utility classes for working with XML in SCIFIO.
