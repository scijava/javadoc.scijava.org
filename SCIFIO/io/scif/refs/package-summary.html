Package io.scif.refs Description

Provides reference management utilities.

The RefManagerService provides a dedicated thread for polling reference queues, looking for available references. The RefManagerService.manage(Object, Object[]) method is used to create and enqueue a new reference. The first parameter is the object to manage, and it must have a corresponding RefProvider that matches the given Object[] parameter list.

The RefProvider is a new plugin type for generating references, and is uniquely defined by the target referent plus an additional set of parameters.

A CleaningRef interface is provided for references that have clean-up operations after being enqueued.
