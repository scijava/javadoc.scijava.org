      • getFormatFromClass

        public <F extends Format> F getFormatFromClass(Class<F> formatClass)
        Description copied from interface: FormatService
        Lookup method for the Format map. Use this method when you want a concrete type reference instead of trying to construct a new Format.

        NB: because SezPoz is used for automatic detection of Formats in SCIFIO, all concrete Format implementations have a zero-parameter constructor. If you manually invoke that constructor and then try to link your Format to an existing context, e.g. via the FormatService.addFormat(Format) method, it will fail if the Format was already discovered. The same principle is true if the context-based constructor is invoked.

        Specified by:
        getFormatFromClass in interface FormatService
        Parameters:
        formatClass - the class of the desired Format
        Returns:
        A reference to concrete class of the queried Format, or null if the Format was not found.
