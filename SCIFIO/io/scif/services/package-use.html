io.scif.services

  • Packages that use io.scif.services 
    Package Description
    io.scif
    Provides the interfaces and default implementations for the components of SCIFIO, and the context for instantiating these components.
    io.scif.services
    Provides a set of Services (singletons within a given context) for working with SCIFIO objects.

    A collection of utility methods to facilitate FilePattern use.

    InitializeService
    A collection of methods for initializing the IO components of SCIFIO (Readers and Writers).
    TranslatorService
    Collection of methods for creating and using appropriate Translators to convert between Metadata types.
