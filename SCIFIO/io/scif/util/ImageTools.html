Class ImageTools

  • java.lang.Object
    • io.scif.util.ImageTools

  • public final class ImageTools
extends Object

      • splitChannels

        public static byte[] splitChannels(byte[] array,
                                   byte[] rtn,
                                   long[] pos,
                                   long[] maxLengths,
                                   int bytes,
                                   boolean reverse,
                                   boolean interleaved,
                                   long planeLength)
        Splits the given multi-channel array into a 2D array. The "reverse" parameter is false if channels are in RGB order, true if channels are in BGR order. If the 'rtn' parameter is not null, the specified channel will be copied into 'rtn'. The 'channelLength' parameter specifies the number of bytes that are expected to be in a single channel. In many cases, this will match the value of 'rtn.length', but specifying it separately allows 'rtn' to be larger than the size of a single channel.
        Parameters:
        pos - - positional axes index of the plane to extract
        maxLengths - - lengths of each split positional axis
        planeLength - - number of bytes in a split out plane
