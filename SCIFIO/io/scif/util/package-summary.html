Package io.scif.util Description

Provides utility classes for common operations on or by SCIFIO components.

Changes since Bio-Formats

  • Because of the split between SCIFIO and OME-XML classes, the Bio-Formats tools class was split into ome.xml.services.DefaultOMEXMLMetadataService and SCIFIOMetadataTools. The OMEXMLMetadata classes can now be found in the OME-Formats project.
