loci.slim

Class SLIMProcessor<T extends RealType<T>>

  • java.lang.Object
    • loci.slim.SLIMProcessor<T>

  • public class SLIMProcessor<T extends RealType<T>>
extends Object
    SLIMProcessor is the main class of the SLIM Curve plugin for ImageJ. It was originally just thrown together to get something working, with some code/techniques borrowed from SLIM Plotter. Parts of this code are ugly and experimental.
    Author:
    Aivar Grislis
