Class AbstractBatchHistogram

  • java.lang.Object

    Direct Known Subclasses:

    public abstract class AbstractBatchHistogram
extends Object
implements BatchHistogram
    Base class for a batch, cumulative histogram for a particular fitted parameter. These histograms have a large number of bins to cover the range of expected values. They accumulate statistics from any number of fitted lifetime images. At any time we can discard outliers using Tukey's Rule http://www.edgarstat.com/tukeys_outliers_help.cfm and zoom in and show the histogram distribution curve.

    Aivar Grislis
