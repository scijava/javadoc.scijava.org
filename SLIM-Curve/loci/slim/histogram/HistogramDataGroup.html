loci.slim.histogram

Class HistogramDataGroup

  • java.lang.Object
    • loci.slim.histogram.HistogramDataGroup

  • public class HistogramDataGroup
extends Object
    Keeps an array of HistogramDataNodes for a given image. Builds the histogram data as appropriate. Handles updates as the fitted results are available. Handles optional auto-ranging.

    Aivar Grislis
