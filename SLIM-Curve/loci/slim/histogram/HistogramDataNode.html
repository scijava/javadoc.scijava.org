      • HistogramDataNode

        public HistogramDataNode(IFittedImage fittedImage,
                         double[][] values)
        Constructor, takes the array of values. This class keeps a separate array of values, rather than just referring to the image values, for two reasons: 1) If the 'chunky pixel' effect is used to draw the image, histogram values will be temporarily incorrect until image is complete. 2) If the image shows colorized grayscale, a la SPCImage, the image values will be incorrect.
