Interface FitSettings

  • All Known Implementing Classes:
    BaseFitSettings, DefaultFitSettings

    public interface FitSettings
    Holds basic FLIM fit settings.

    The intention here is to provide a basic set of fit settings that every FLIM fitting implementation ought to need. If it turns out some settings are irrelevant to a given fitting implementation they can simply be ignored.

    Author:
    Aivar Grislis
