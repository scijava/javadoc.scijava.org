      • launch

        public void launch(String... args)
        Description copied from interface: Gateway
        Perform launch operations associated with this gateway.

        Typical operations might include:

        • Handle the given command line arguments using the ConsoleService.
        • Execute registered main classes of the MainService.
        • Display the default user interface using the UIService.
        • In some circumstances (e.g., when running headless), dispose the context after launch operations are complete.
        Specified by:
        launch in interface Gateway
        Parameters:
        args - The arguments to pass to the application.
