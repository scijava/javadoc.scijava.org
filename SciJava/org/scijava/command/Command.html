Interface Command

  • All Superinterfaces:
    Runnable, SciJavaPlugin
    All Known Implementing Classes:
    ClearRecent, Close, CloseAll, CommandFinder, ContextCommand, DialogPromptDemo, DumpStack, DynamicCommand, FileScriptBatchProcessor, InspectObjects, InteractiveCommand, MATLABOptions, MenuScriptBatchProcessor, ModuleBatchProcessor, ModuleCommand, OpenFile, OptionsLookAndFeel, OptionsPlugin, ParamAssignOrder, RunScript, ShowConsole, ShowSubscribers, SystemInformation, TextDisplayTest, UnimplementedCommand, WatchEvents, WindowsTileVertical

    public interface Command
extends SciJavaPlugin, Runnable
    Command is a plugin that is executable. A Command can be executed as a Module by wrapping it in an instance of CommandModule.

    Commands discoverable at runtime must implement this interface and be annotated with @Plugin with attribute Plugin.type() = Command.class. While it possible to create a command merely by implementing this interface, it is encouraged to instead extend ContextCommand (or in some cases ModuleCommand, DynamicCommand or InteractiveCommand) for convenience.
