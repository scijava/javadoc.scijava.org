Interface ConsoleArgument

  • All Superinterfaces:

    public interface ConsoleArgument
extends HandlerPlugin<LinkedList<String>>
    A plugin which extends an application's command line argument handling.

    Console argument plugins discoverable at runtime must implement this interface and be annotated with @Plugin with attribute Plugin.type() = ConsoleArgument.class. While it is possible to create an console argument plugin merely by implementing this interface, it is encouraged to instead extend AbstractConsoleArgument, for convenience.
