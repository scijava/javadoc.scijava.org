        default Location resolve(URI uri)
                  throws URISyntaxException
        Resolves the given URI to a location. If the scheme part of the URI is null the path component is resolved as a local file.
        Parameters:
        uri - the uri to resolve
        Returns:
        the resolved Location or null if no resolver could be found.
        Throws:
        URISyntaxException - if the URI is malformed
