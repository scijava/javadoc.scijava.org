Class BuildEnvironment

  • java.lang.Object
    • org.scijava.minimaven.BuildEnvironment

  • public class BuildEnvironment
extends Object
    The build environment of MiniMaven.

    This class contains settings and methods that are not specific to a single MavenProject, but rather specific to a tree of projects.

    Author:
    Johannes Schindelin
