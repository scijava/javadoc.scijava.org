Class MavenProject

  • java.lang.Object
    • org.scijava.minimaven.MavenProject
  • All Implemented Interfaces:
    Comparable<MavenProject>

    public class MavenProject
extends Object
implements Comparable<MavenProject>
    This class represents a parsed pom.xml file.

    Every pom.xml file is parsed into an instance of this class; the tree of projects shares a BuildEnvironment instance.

    Author:
    Johannes Schindelin
