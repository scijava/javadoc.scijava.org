Class MacOSPlatform

  • java.lang.Object
    • org.scijava.platform.AbstractPlatform
      • org.scijava.plugins.platforms.macos.MacOSPlatform
  • All Implemented Interfaces:
    Disposable, Platform, SciJavaPlugin, SingletonPlugin

    public class MacOSPlatform
extends AbstractPlatform
    A platform implementation for handling Apple macOS platform issues:
    • Application events are rebroadcast as ImageJ events.
    • macOS screen menu bar is enabled.
    • Special screen menu bar menu items are handled.
    Author:
    Curtis Rueden
