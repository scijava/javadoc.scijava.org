      • proxy

        public static matlabcontrol.MatlabProxy proxy()
        A running MATLAB instance can only have one proxy active at a time. Attempting to create multiple proxies will cause additional instances of MATLAB to be spawned - which we would like to avoid. Thus this method will cache a proxy and return it if it is still connected. If not, a new proxy will be generated with the default configuration:
        • hidden = true
        • multithreaded = true
        • license = null
        Returns:
        An active MatlabProxy.
