    public abstract class AbstractInputHarvesterPlugin<P,W>
extends AbstractInputHarvester<P,W>
implements PreprocessorPlugin
    AbstractInputHarvesterPlugin is an InputHarvester that implements the PreprocessorPlugin interface. It is intended to be extended by UI-specific implementations such as SwingInputHarvester.

    The input harvester will first check whether the default UI matches that of its implementation; for example, the Swing-based input harvester plugin will only harvest inputs if the Swing UI is currently the default one.
