Class AWTInputEventDispatcher

  • java.lang.Object
    • org.scijava.ui.awt.AWTInputEventDispatcher
  • All Implemented Interfaces:
    KeyListener, MouseListener, MouseMotionListener, MouseWheelListener, EventListener

    public class AWTInputEventDispatcher
extends Object
implements KeyListener, MouseListener, MouseMotionListener, MouseWheelListener
    Rebroadcasts AWT InputEvents as ImageJ InputEvents, translating KeyEvents into KyEvents, and MouseEvents into MsEvents.
    Author:
    Curtis Rueden, Grant Harris
