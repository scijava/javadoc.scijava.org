                                         DialogPrompt.MessageType msg,

        Description copied from interface: UserInterface
        Creates a dialog prompter.
        Parameters:
        message - The message in the dialog itself.
        title - The title of the dialog.
        msg - The type of message. This typically is rendered as an icon next to the message. For example, DialogPrompt.MessageType.WARNING_MESSAGE typically appears as an exclamation point.
        option - The choices available when dismissing the dialog. These choices are typically rendered as buttons for the user to click.
        Returns:
        The newly created DialogPrompt object, or null if not applicable.
