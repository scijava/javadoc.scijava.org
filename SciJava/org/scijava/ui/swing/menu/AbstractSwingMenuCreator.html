    Direct Known Subclasses:

    public abstract class AbstractSwingMenuCreator<T>
extends AbstractMenuCreator<T,JMenu>
    Populates a Swing menu structure with menu items from a ShadowMenu.

    This class is called AbstractSwingMenuCreator rather than simply AbstractMenuCreator to avoid having multiple classes with the same name in different packages (e.g., imagej.menu.AbstractMenuCreator and imagej.ui.swing.menu.AbstractMenuCreator).
